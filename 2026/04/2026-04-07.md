## 📝 상세 요약

📄 *1. Hierarchical Planning with Latent World Models*

🔗 https://arxiv.org/abs/2604.03208

🏢 *NYU, Meta AI (Yann LeCun 포함)*

🔬 *메소드*: 서로 다른 시간 스케일에서 잠재 세계 모델을 학습하고, 이들 스케일 간 계층적 계획을 수행하는 모델 예측 제어 방법론을 제안합니다. 단일 레벨 세계 모델 대신 다중 시간 해상도를 활용하여 장기간 추론을 가능하게 하면서도 추론 시 계획 복잡도를 크게 줄입니다. 이 접근법은 다양한 잠재 세계 모델 아키텍처와 도메인에 적용 가능한 모듈형 계획 추상화 역할을 합니다.

💡 *컨트리뷰션*: 예측 오차 누적과 지수적으로 증가하는 탐색 공간 문제를 해결하여 장기간 제어에서 학습된 세계 모델의 성능을 크게 향상시켰습니다.

🧪 *실험*: 실제 로봇 pick-&-place 작업에서 70% 성공률을 달성(단일 레벨 모델은 0%), 물리 시뮬레이션 환경(push manipulation, maze navigation)에서 최대 4배 적은 계산 비용으로 더 높은 성공률을 기록했습니다.

⭐ **Yann LeCun이 참여한 연구로, 다중 시간 스케일 계층적 계획을 통해 장기간 로봇 제어 성능을 획기적으로 향상시킨 world model 기반 제어 방법론**


📄 *2. HeteroHub: An Applicable Data Management Framework for Heterogeneous Multi-Embodied Agent System*

🔗 https://arxiv.org/abs/2603.28010

🏢 *홍콩과학기술대학교*

🔬 *메소드*: 이질적인 멀티 에이전트 시스템을 위한 데이터 중심 관리 프레임워크를 제안합니다. 정적 메타데이터, 작업 맞춤형 훈련 코퍼스, 실시간 데이터 스트림을 통합하여 관리합니다. 작업 인식 모델 훈련, 상황 인식 실행, 실제 피드백 기반 폐쇄 루프 제어를 지원하는 통합 데이터 관리 인프라를 구축했습니다.

💡 *컨트리뷰션*: 기존 프레임워크에서 부족했던 이질적 멀티 에이전트 시스템의 실제 배포를 위한 통합 데이터 관리 인프라를 최초로 제공했습니다.

🧪 *실험*: 실제 데모에서 여러 embodied AI 에이전트가 복잡한 작업을 성공적으로 협력 수행했으며, 확장 가능하고 유지보수 가능한 embodied AI 시스템 구축 가능성을 입증했습니다.

⭐ **이질적인 다중 로봇 시스템의 실제 배포를 위한 최초의 통합 데이터 관리 프레임워크로 확장 가능한 embodied AI 시스템 구축의 기반 제공**


📄 *3. Multi-View Video Diffusion Policy: A 3D Spatio-Temporal-Aware Video Action Model*

🔗 https://arxiv.org/abs/2604.03181

🏢 *중국과학원*

🔬 *메소드*: 3D 공간-시간 상태를 동시에 모델링하는 다시점 비디오 확산 정책을 제안합니다. 멀티뷰 히트맵 비디오와 RGB 비디오를 동시에 예측하여 비디오 사전훈련과 액션 파인튜닝의 표현 형식을 정렬하고, 로봇이 취해야 할 액션과 환경이 어떻게 반응할지를 모두 명시합니다. 기존 2D 기반 정책들이 놓친 3D 구조와 시간적 진화를 모두 고려합니다.

💡 *컨트리뷰션*: 단 10개의 시연 궤적만으로도 복잡한 실제 작업을 성공적으로 수행할 수 있는 데이터 효율적인 조작 정책을 실현했습니다.

🧪 *실험*: Meta-World와 실제 로봇 플랫폼에서 기존 비디오 예측 기반, 3D 기반, 비전-언어-액션 모델들을 일관되게 능가하며 데이터 효율적 멀티태스크 조작에서 새로운 최고 성능을 달성했습니다.

⭐ **단 10개 시연으로 복잡한 실제 로봇 작업을 수행하는 3D 공간-시간 인식 비디오 확산 정책으로 데이터 효율적 조작의 새로운 최고 성능 달성**


📄 *4. Revealing Physical-World Semantic Vulnerabilities: Universal Adversarial Patches for Infrared Vision-Language Models*

🔗 https://arxiv.org/abs/2604.03117

🏢 *국방과학기술대학교(중국)*

🔬 *메소드*: 적외선 비전-언어 모델을 대상으로 하는 Universal Curved-Grid Patch(UCGP) 프레임워크를 제안합니다. Curved-Grid Mesh 매개변수화로 연속적이고 배포 가능한 패치를 생성하며, 부공간 이탈, 토폴로지 파괴, 은닉성을 촉진하는 통합 표현 기반 목표 함수를 사용합니다. Meta Differential Evolution과 EOT 증강 TPS 변형 모델링을 통해 실제 배포 상황에서의 강건성을 향상시켰습니다.

💡 *컨트리뷰션*: 기존 RGB 기반 공격과 달리 적외선 VLM의 시각 표현 공간을 직접 교란하여 교차 모달 의미 정렬을 약화시키는 새로운 접근법을 제시했습니다.

🧪 *실험*: 다양한 IR-VLM 아키텍처에서 일관되게 의미 이해를 손상시켰으며, 교차 모델 전이성, 교차 데이터셋 일반화, 실제 물리적 효과성, 방어에 대한 강건성을 모두 입증했습니다.

⭐ **적외선 비전-언어 모델의 취약점을 최초로 체계적으로 분석하고 실제 물리 환경에서 효과적인 범용 적대적 패치 공격 방법 개발**


📄 *5. V2X-QA: A Comprehensive Reasoning Dataset and Benchmark for Multimodal Large Language Models in Autonomous Driving Across Ego, Infrastructure, and Cooperative Views*

🔗 https://arxiv.org/abs/2604.02710

🏢 *위스콘신대학교 매디슨 캠퍼스*

🔬 *메소드*: 차량, 인프라, 협력 관점을 포괄하는 V2X-QA 벤치마크를 구축했습니다. 뷰 분리 평가 프로토콜을 통해 차량 단독, 인프라 단독, 협력 주행 조건에서 통제된 비교를 가능하게 하며, 12개 작업 분류체계로 인식, 예측, 추론 및 계획을 평가합니다. 명시적 뷰 라우팅과 관점별 LoRA 전문가를 가진 V2X-MoE 기준 모델도 제안했습니다.

💡 *컨트리뷰션*: 기존 자기중심적 벤치마크를 넘어 인프라 중심 및 협력 주행 조건에서 MLLM 성능을 체계적으로 평가할 수 있는 최초의 포괄적 데이터셋을 구축했습니다.

🧪 *실험*: 10개 대표적인 최신 모델 평가 결과, 관점 접근성이 성능에 상당한 영향을 미치며, 인프라 측면 추론이 의미 있는 거시적 교통 이해를 지원함을 확인했습니다. 협력 추론은 여전히 도전적인 과제로 나타났습니다.

⭐ **자율주행에서 차량-인프라-협력 관점을 포괄하는 최초의 멀티모달 LLM 벤치마크로 연결 자율주행의 다관점 추론 연구 기반 마련**

📌 *금주 트렌드 요약*
계층적 구조와 다중 관점 통합이 핵심 트렌드로 부상하고 있으며, 특히 시간적-공간적 다중 스케일 모델링과 멀티뷰 접근법이 로봇 제어와 자율주행 성능을 크게 향상시키고 있습니다. 데이터 효율성이 중요한 실용적 과제로 인식되면서, 소수의 시연만으로도 복잡한 실제 작업을 수행할 수 있는 방법론들이 주목받고 있습니다. 동시에 실제 배포를 위한 통합 프레임워크와 보안 취약점 분석도 중요한 연구 영역으로 확장되고 있습니다.
