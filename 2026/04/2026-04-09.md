## 📝 상세 요약

📄 *1. Gym-Anything: Turn any Software into an Agent Environment*

🔗 https://arxiv.org/abs/2604.06126

🏢 *Pranjal Aggarwal, Graham Neubig, Sean Welleck*

🔬 *메소드*: 임의의 소프트웨어를 상호작용 가능한 컴퓨터 사용 환경으로 자동 변환하는 멀티에이전트 프레임워크를 제안. 코딩 에이전트가 설정 스크립트를 작성하고 실제 데이터를 다운로드하여 소프트웨어를 구성하는 동안, 독립적인 감사 에이전트가 품질 체크리스트에 따라 환경 설정을 검증함.

💡 *컨트리뷰션*: 기존 제한적인 소프트웨어 환경 대신 미국 GDP 데이터 기반의 200개 소프트웨어 애플리케이션을 포함하는 10K 이상의 장기 작업 벤치마크 CUA-World를 구축.

🧪 *실험*: 500+ 스텝을 요구하는 CUA-World-Long 벤치마크에서 2B 비전-언어 모델이 2배 큰 모델을 능가하는 성능을 보임. Gemini-3-Flash 성능을 11.5%에서 14.0%로 향상시킴.

⭐ **복잡한 소프트웨어를 AI 학습 환경으로 자동 변환하는 멀티에이전트 시스템으로 실용적 컴퓨터 작업 벤치마크를 대규모 구축**


📄 *2. MPM: Mutual Pair Merging for Efficient Vision Transformers*

🔗 https://arxiv.org/abs/2604.05718

🏢 *Simon Ravé, Pejman Rasti, David Rousseau*

🔬 *메소드*: 의미적 분할을 위한 학습 없는 토큰 집계 모듈로, 코사인 공간에서 상호 최근접 이웃 쌍을 형성하고 평균화한 후 gather 기반 재구성을 가능하게 하는 병합 맵을 기록. 학습 가능한 파라미터나 연속적인 압축 노브 없이 이산적인 삽입 스케줄로 속도-정확도 트레이드오프를 조절.

💡 *컨트리뷰션*: 기존 토큰 축소 방법들이 분류에 집중한 반면, 픽셀 단위 밀집 특징 재구성이 필요한 의미적 분할에서 실제 지연 시간 개선을 달성.

🧪 *실험*: ADE20K에서 ViT-Tiny 모델이 Raspberry Pi 5에서 최대 60% 지연 시간 감소, H100 GPU에서 FlashAttention-2 사용 시 최대 20% 처리량 증가를 달성하면서 mIoU 저하를 3% 미만으로 유지.

⭐ **의미적 분할을 위한 간단하고 학습 없는 토큰 병합 방법으로 실제 하드웨어에서 상당한 속도 향상을 달성**


📄 *3. A1: A Fully Transparent Open-Source, Adaptive and Efficient Truncated Vision-Language-Action Model*

🔗 https://arxiv.org/abs/2604.05672

🏢 *Kaidong Zhang, Jian Zhang, Rongtao Xu 외 다수*

🔬 *메소드*: 저비용 고처리량 추론을 위한 완전 오픈소스 VLA 프레임워크로, 중간 VLM 레이어에서 액션 일관성을 모니터링하여 조기 종료를 트리거하는 예산 인식 적응 추론 방식을 도입. Inter-Layer Truncated Flow Matching을 통해 레이어 간 노이즈 제거를 웜 스타트하여 적은 반복으로 정확한 액션 생성.

💡 *컨트리뷰션*: VLM 백본과 액션 헤드를 공동 가속화하는 전체 추론 파이프라인 최적화와 완전한 훈련 스택 공개로 종단간 재현성 제공.

🧪 *실험*: LIBERO, VLABench 시뮬레이션과 Franka, AgiBot 실제 로봇에서 최대 72% 에피소드별 지연 시간 감소, 76.6% 백본 연산 감소 달성. RoboChallenge에서 29.00% 평균 성공률로 pi0(28.33%), X-VLA(21.33%)를 능가.

⭐ **저비용 실시간 로봇 제어를 위해 VLA 모델의 전체 추론 파이프라인을 최적화한 완전 오픈소스 프레임워크**


📄 *4. SnapFlow: One-Step Action Generation for Flow-Matching VLAs via Progressive Self-Distillation*

🔗 https://arxiv.org/abs/2604.05656

🏢 *Wuyang Luan, Junhui Li, Weiguang Zhao, Wenjian Zhang, Tieru Wu, Rui Ma*

🔬 *메소드*: 플로우 매칭 VLA의 다단계 노이즈 제거를 단일 순전파로 압축하는 플러그앤플레이 자기 증류 방법. 표준 플로우 매칭 샘플과 모델 자체의 주변 속도 예측으로부터 계산된 2단계 오일러 단축 속도를 타겟으로 하는 일관성 샘플을 혼합하여 조건부 속도로 인한 궤적 드리프트 방지.

💡 *컨트리뷰션*: 외부 교사 모델이나 아키텍처 변경 없이 단일 GPU에서 12시간 내 훈련이 가능하며, 제로 초기화된 타겟-시간 임베딩으로 로컬 속도 추정과 글로벌 원스텝 생성을 단일 아키텍처 내에서 전환.

🧪 *실험*: pi0.5(3B)에서 LIBERO 4개 슈트 40개 태스크에 대해 98.75% 평균 성공률로 10단계 교사 모델(97.75%)과 동등한 성능을 달성하면서 9.6배 노이즈 제거 가속화, 종단간 지연시간을 274ms에서 83ms로 단축. SmolVLA(500M)에서 MSE 8.3% 감소와 3.56배 가속화.

⭐ **플로우 매칭 VLA의 다단계 노이즈 제거를 원스텝으로 압축하여 성능 유지하면서 대폭적인 추론 가속화를 달성**


📄 *5. Grounding Hierarchical Vision-Language-Action Models Through Explicit Language-Action Alignment*

🔗 https://arxiv.org/abs/2604.05614

🏢 *Theodor Wulff, Federico Tavella, Rahul Singh Maharjan, Manith Adikari, Angelo Cangelosi*

🔬 *메소드*: 계층적 VLA 모델의 하위 태스크 설명을 시각적 관찰과 액션 공간에 명시적으로 그라운딩하는 훈련 프레임워크 제안. 생성된 언어와 해당 액션 궤적 간 정렬을 평가하는 대조 모델을 사용하여 언어-궤적 쌍을 직접 순위화하고 오프라인 선호도 학습으로 그라운딩을 개선.

💡 *컨트리뷰션*: 기존 연구에서 고려되지 않았던 언어와 액션 모달리티 간의 명시적 정렬을 훈련 중에 통합하여 로봇의 투명성과 인간-로봇 협업 효과성 향상.

🧪 *실험*: 인간 언어 주석 궤적의 벤치마크 데이터셋인 LanguageTable에서 완전 지도 파인튜닝과 비교 가능한 성능을 달성하면서 비용이 많이 드는 데이터 주석의 필요성을 최소화.

⭐ **로봇 투명성 향상을 위해 계층적 VLA 모델의 언어와 액션 간 명시적 정렬을 학습하는 대조 기반 프레임워크**

📌 *금주 트렌드 요약*
VLA(Vision-Language-Action) 모델의 실용적 배포를 위한 효율성 최적화가 주요 트렌드로 나타남. 추론 속도 개선을 위한 다양한 접근법(토큰 축소, 조기 종료, 원스텝 생성)과 함께 실제 하드웨어 환경에서의 성능 검증이 강조됨. 동시에 로봇의 투명성과 신뢰성 향상을 위한 언어-액션 정렬 연구와 대규모 실용적 벤치마크 구축이 병행되고 있음.
