## 📝 상세 요약

📄 *1. One Token Per Frame: Reconsidering Visual Bandwidth in World Models for VLA Policy*

🔗 https://arxiv.org/abs/2605.07931

🏢 *Zuojin Tang, Shengchao Yuan, Xiaoxin Bai, Zhiyuan Jin, De Ma, Gang Pan, Bin Liu*

🔬 *메소드*: 기존 Vision-Language-Action 모델이 프레임당 고해상도 비주얼 정보를 사용하는 것과 달리, Adaptive Attention Pooling을 통해 각 뷰를 프레임당 단일 의미 토큰으로 압축하는 OneWM-VLA를 제안합니다. 기존의 별도 디코더를 통한 연결 대신 단일 flow-matching 목적함수 하에서 잠재 스트림과 액션 궤적을 통합적으로 생성합니다.

💡 *컨트리뷰션*: 프레임당 비주얼 대역폭을 단일 토큰으로 대폭 줄이면서도 장기 호라이즌 성능을 유지할 수 있음을 증명했으며, 제한된 적응 예산 하에서 표현 학습과 액션 결합을 효율적으로 개선했습니다.

🧪 *실험*: 2B 백본에 14.71M LoRA 파라미터로 학습하여 MetaWorld MT50에서 평균 성공률을 47.9%에서 61.3%로 향상, LIBERO-Long에서 95.6% 달성(π_0 대비 85.2%), 실제 Piper 로봇 팔에서 Fold Cloth 작업 성공률을 20.0%에서 60.0%로 개선했습니다.

⭐ **프레임당 단일 토큰으로 비주얼 정보를 압축하면서도 VLA 모델의 장기 호라이즌 성능을 크게 향상시킨 효율적인 세계 모델**


📄 *2. ForgeVLA: Federated Vision-Language-Action Learning without Language Annotations*

🔗 https://arxiv.org/abs/2605.07474

🏢 *Yuhao Zhou, Yunpeng Zhu, Yang Zhou, Jindi Lyu, Jian Lan, Zhangyuan Wang, Dan Si, Thomas Seidl, Qing Ye, Jiancheng Lyu*

🔬 *메소드*: 분산된 비전-액션 쌍으로부터 중앙 집중식 데이터 수집이나 수동 어노테이션 없이 VLA 모델을 학습하는 연합 학습 프레임워크를 제안합니다. 각 클라이언트는 embodied instruction classifier를 통해 비전-액션 쌍을 사전 정의된 명령어 세트에 매핑하여 누락된 언어 모달리티를 복원합니다.

💡 *컨트리뷰션*: 원시 데이터 중앙화나 수동 어노테이션 없이 연합 VLA 학습을 가능하게 했으며, 비전-언어 특징 붕괴 문제를 해결하는 클라이언트 측 대조 계획 손실과 서버 측 적응적 집계 전략을 도입했습니다.

🧪 *실험*: 여러 벤치마크에서 ForgeVLA가 다른 베이스라인 대비 상당한 성능 향상을 보였으며, 각 컴포넌트의 기여도를 검증하는 ablation study를 통해 제안 방법의 유효성을 입증했습니다.

⭐ **언어 어노테이션 없이 분산된 로봇 데이터로부터 연합 학습을 통해 VLA 모델을 효율적으로 훈련하는 혁신적 프레임워크**


📄 *3. Search-based Robustness Testing of Laptop Refurbishing Robotic Software*

🔗 https://arxiv.org/abs/2605.07530

🏢 *Erblin Isaku, Hassan Sartaj, Shaukat Ali, Malaika Din Hashmi, Francois Picard (Danish Technological Institute)*

🔬 *메소드*: 랩톱 재생 로봇의 객체 검출 모델 견고성을 테스트하기 위해 NSGA-II 다목적 최적화를 활용한 PROBE라는 검색 기반 테스트 접근법을 제안합니다. 최소한의 국소적 섭동을 통해 모델 실패를 유도하면서 국소화/신뢰도와 섭동 크기를 동시에 최적화합니다.

💡 *컨트리뷰션*: 실제 산업용 로봇 소프트웨어에 대한 체계적인 견고성 테스트 방법론을 제시했으며, 메타모픽 관계를 통한 추가적인 모델 안정성 평가 방법을 도입했습니다.

🧪 *실험*: PROBE는 랜덤 서치 대비 3-7배 더 효과적으로 실패 유도 섭동을 생성하면서도 더 작은 섭동 크기를 요구했으며, 생성된 섭동이 모델 간 전이 가능함을 확인했습니다.

⭐ **실제 랩톱 재생 로봇의 객체 검출 모델 견고성을 체계적으로 테스트하는 다목적 최적화 기반 검색 방법론**


📄 *4. EggHand: A Multimodal Foundation Model for Egocentric Hand Pose Forecasting*

🔗 https://arxiv.org/abs/2605.07642

🏢 *Jaeyoung Choi, Hyeondong Kim, Yujin Kim, Daehee Park*

🔬 *메소드*: 1인칭 시점에서 미래 3D 손 포즈 시퀀스를 예측하기 위해 Vision-Language-Action 모델의 액션 디코더와 1인칭 비디오-텍스트 인코더를 결합한 파운데이션 모델 기반 프레임워크를 제안합니다. 손동작의 구조적 시간 역학과 시점 인식 문맥 정보를 통합하여 다중모달 의미 추론과 동적 모션 모델링을 통합합니다.

💡 *컨트리뷰션*: 신체 포즈나 외부 추적 없이도 모션, 문맥, 고수준 의도를 통합 추론할 수 있는 방법을 제시했으며, 언어 기반 태스크 프롬프트를 통한 제어 가능한 예측 기능을 구현했습니다.

🧪 *실험*: EgoExo4D 데이터셋에서 새로운 최고 성능을 달성했으며, 심한 자기 모션 하에서도 견고성을 유지하고 언어 기반 태스크 프롬프트를 통한 제어 가능한 예측을 성공적으로 구현했습니다.

⭐ **1인칭 시점의 복잡한 손동작을 다중모달 파운데이션 모델로 정확하게 예측하는 혁신적인 hand pose forecasting 시스템**


📄 *5. PathPainter: Transferring the Generalization Ability of Image Generation Models to Embodied Navigation*

🔗 https://arxiv.org/abs/2605.07496

🏢 *Yijin Wang, Yuru Tian, Xijie Huang, Weiqi Gai, Mo Zhu, Xin Zhou, Yuze Wu, Fei Gao*

🔬 *메소드*: Bird's-eye-view 이미지를 글로벌 prior로 활용하는 내비게이션 시스템으로, 이미지 생성 모델을 사용하여 자연어로부터 인간 의도를 해석하고 목표 지점을 식별하며 통행 가능성 마스크를 생성합니다. 실행 중에는 교차 뷰 로컬라이제이션을 통해 로봇의 odometry를 BEV 맵과 정렬하여 장기 드리프트를 완화합니다.

💡 *컨트리뷰션*: 파운데이션 모델의 세계 이해 능력을 구현된 내비게이션에 전이하는 방법을 제시했으며, 기존 이미지 생성 모델의 강력한 일반화 능력을 로봇이 활용할 수 있게 했습니다.

🧪 *실험*: 기존 로컬 모션 플래너만으로 UAV가 160미터의 실외 장거리 내비게이션 작업을 성공적으로 완료했으며, 광범위한 벤치마크 실험을 통해 제안 방법의 유효성을 검증했습니다.

⭐ **이미지 생성 모델의 일반화 능력을 활용하여 자연어 명령으로 장거리 로봇 내비게이션을 가능하게 하는 혁신적 시스템**

📌 *금주 트렌드 요약*
VLA(Vision-Language-Action) 모델의 효율성 개선과 실제 응용에 대한 연구가 주를 이루며, 특히 데이터 효율성과 연합 학습을 통한 스케일링 방법론이 주목받고 있습니다. 파운데이션 모델의 일반화 능력을 로봇 태스크에 전이하는 연구와 실제 산업 환경에서의 견고성 확보가 중요한 트렌드로 부상하고 있습니다. 1인칭 시점 이해와 장기 호라이즌 예측 능력 향상이 embodied AI의 핵심 과제로 인식되고 있습니다.
