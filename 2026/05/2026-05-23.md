## 📝 상세 요약

📄 *1. GesVLA: Gesture-Aware Vision-Language-Action Model Embedded Representations*

🔗 https://arxiv.org/abs/2605.22812

🏢 *Wenxuan Guo 외 9명* (중국 베이징대학교 외)

🔬 *메소드*: 기존 VLA 모델이 텍스트 명령만 사용해 복잡한 장면에서 공간적 모호성을 해결하지 못하는 문제를 해결하기 위해 제스처를 병렬 명령 모달리티로 도입한 GesVLA를 제안합니다. 제스처 특징을 잠재 공간에 직접 인코딩하여 고수준 추론과 저수준 행동 생성에 모두 참여하도록 하고, 듀얼 VLM 아키텍처를 통해 제스처 표현과 행동 정책 간의 긴밀한 결합을 구현했습니다. 실제 장면 이미지에 손 모델을 렌더링하여 확장 가능한 제스처 데이터 생성 파이프라인을 구축하고 2단계 훈련 전략을 사용합니다.

💡 *컨트리뷰션*: 복잡하고 혼잡한 환경에서 여러 유사한 객체가 있을 때 발생하는 공간적 모호성을 해결하기 위해 제스처를 VLA 모델에 통합한 최초의 연구입니다.

🧪 *실험*: 블록 조작 작업, 제품 선택, 농산물 선택 등 실제 로봇 작업에서 평가했으며, 제스처 도입으로 대상 객체 grounding 정확도와 인간-로봇 상호작용 효율성이 일관되게 개선됨을 확인했습니다.

⭐ **제스처를 VLA 모델에 통합하여 복잡한 환경에서 공간적 모호성을 해결하고 인간-로봇 상호작용 효율성을 크게 향상시킨 연구**


📄 *2. Pre-VLA: Preemptive Runtime Verification for Reliable Vision-Language-Action and World-Model Rollouts*

🔗 https://arxiv.org/abs/2605.22446

🏢 *Zhen Sun 외 8명* (중국)

🔬 *메소드*: VLA 모델과 생성 월드 모델의 불확실한 행동 생성 문제를 해결하기 위해 물리적 실행이나 월드 모델 시뮬레이션 전에 선제적 행동 유효성 평가를 수행하는 Pre-VLA를 제안합니다. 효율적인 멀티모달 백본과 modality-aware 풀링, 그리고 경량 듀얼 브랜치 헤드를 활용하여 안전성 신뢰도와 critic 기반 advantage 점수를 예측합니다. Focal 분류, advantage 회귀, 소프트 임계값 보정을 결합한 멀티태스크 목적 함수로 훈련하고, 배포 시 듀얼 모드 선제적 리샘플링 스케줄러를 사용합니다.

💡 *컨트리뷰션*: VLA 모델의 행동 품질을 실행 전에 평가하고 필터링하는 런타임 검증 아키텍처를 도입하여 물리적 실패와 오해를 유발하는 월드 모델 롤아웃을 방지합니다.

🧪 *실험*: LIBERO 벤치마크에서 RynnVLA-002 대비 평균 폐루프 성공률을 30.79%에서 37.62%로 향상시켰으며, 행동 청크당 평균 183.9ms의 검증 시간을 달성했습니다.

⭐ **VLA 모델의 행동을 실행 전에 검증하여 안전성과 성공률을 크게 향상시키는 선제적 런타임 검증 프레임워크**


📄 *3. Beyond Pixels: Learning Invariant Rewards for Real-World Robotics From a Few Demonstrations*

🔗 https://arxiv.org/abs/2605.22123

🏢 *Tengye Xu 외 8명* (홍콩대학교 외)

🔬 *메소드*: 기존 비전 기반 보상 모델이 특정 픽셀 분포를 암기하여 일반화에 실패하는 문제를 해결하기 위해, 5개 정도의 적은 시연으로부터 불변 심볼릭 보상 함수를 학습하는 프레임워크를 제안합니다. 시각적 특징 맞춤에서 벗어나 행동 불변량 발견으로 패러다임을 전환하며, 작업 수준의 전략과 물리적 제약을 인코딩하는 구조적 보상 공식화와 온라인 상호작용 없이 시연에서 불변량을 추출하는 하이브리드 심볼릭-수치 절차를 결합했습니다.

💡 *컨트리뷰션*: 시각적 변화에 강건한 행동 불변량을 발견하여 다양한 시각적 인스턴스화에서도 일정한 작업 수준 속성을 유지하는 보상 함수 학습 방법을 제시합니다.

🧪 *실험*: Meta-World 8개 작업과 Franka 조작 3개 작업에서 기존 방법 대비 더 강한 프로세스 정렬과 정책 롤아웃 순위 능력을 보였으며, 실제 환경에서 위치, 시점, 객체 변화에 대해 제로샷 일반화를 달성했습니다.

⭐ **적은 시연으로부터 시각적 변화에 강건한 불변 심볼릭 보상 함수를 학습하여 실제 환경 변화에 제로샷 일반화를 달성한 연구**


📄 *4. CoRMA: Contrastive RMA for Contact-Rich Meta-Adaptation*

🔗 https://arxiv.org/abs/2605.22082

🏢 *Wentian Wang 외 8명*

🔬 *메소드*: 힘이 지배적인 조립 작업을 위해 RMA를 수정한 대조적 로봇 모터 적응(CoRMA) 프레임워크를 제안합니다. 원시 시뮬레이터 매개변수 적응을 접촉 시작, 측면 결합, 유도 전환, 접촉 방향, 잼밍을 설명하는 컴팩트한 6D 시뮬레이터 전용 의미적 접촉 컨텍스트로 대체했습니다. 배포 가능한 인과관계 Transformer 어댑터가 의미적 회귀와 힘 체제 대조 목적을 사용하여 힘, 고유수용감각, 행동 이력으로부터 이 컨텍스트를 온라인으로 추론합니다.

💡 *컨트리뷰션*: 시연, 특권 입력, 기울기 업데이트 없이 에피소드 내 적응을 가능하게 하는 의미적 접촉 추론을 적응 인터페이스로 도입했습니다.

🧪 *실험*: Isaac Lab/Isaac Sim 5.0에서 PegInsert, GearMesh, NutThread 작업과 실제 Marvin 로봇 팔에서 평가했으며, 높은 시뮬레이션 성공률을 보이지만 하드웨어에서 크게 저하되는 FORGE 베이스라인 대비 제어된 목표 포즈 노이즈 하에서 더 높은 실제 성공률을 유지했습니다.

⭐ **접촉이 많은 조립 작업에서 의미적 접촉 컨텍스트를 통해 시뮬레이션과 실제 환경 간의 성능 격차를 줄인 메타 적응 프레임워크**


📄 *5. From Abstraction to Instantiation: Learning Behavioral Representation for Vision-Language-Action Model*

🔗 https://arxiv.org/abs/2605.22671

🏢 *Bing Hu 외 6명* (중국 중산대학교 외)

🔬 *메소드*: VLA 모델이 분포 변화 하에서 성능 저하를 겪는 문제를 해결하기 위해 시간적으로 일관된 행동 표현을 학습하는 BehaviorVLA 프레임워크를 제안합니다. 인과관계 Mamba 기반 아키텍처를 사용하여 장기간 궤적 정보를 통합 행동 표현으로 집계하는 Visuomotor Behavior Encoder(VBE)와, 작업 수준 사전 지식을 실시간 실행 진행 상황과 동적으로 정렬하여 이 표현을 정확한 행동으로 디코딩하는 Phase-conditioned Behavior Decoder(PBD)로 구성됩니다.

💡 *컨트리뷰션*: 단기간 시간적 단편화와 정적 실행 정렬 문제를 해결하여 복잡한 시나리오에서도 일관된 행동을 생성하는 시간적으로 일관된 행동 표현 학습 방법을 제시했습니다.

🧪 *실험*: RoboTwin 2.0, LIBERO, CALVIN에서 각각 58%, 98%, 4.36(평균 길이)의 최고 수준 성공률을 달성했으며, 실제 sim-to-real 전이에서 OpenVLA-OFT와 동등한 성능을 50% 시연 데이터만으로 달성했습니다.

⭐ **장기간 궤적을 통합한 시간적 일관성 있는 행동 표현을 통해 분포 변화에 강건하고 데이터 효율적인 VLA 모델을 구현한 연구**

📌 *금주 트렌드 요약*

VLA(Vision-Language-Action) 모델의 실용성과 강건성 향상이 주요 트렌드로, 제스처/멀티모달 입력 통합, 런타임 검증을 통한 안전성 보장, 그리고 시각적/분포 변화에 강건한 표현 학습이 핵심 연구 방향으로 부상하고 있습니다. 또한 적은 데이터로 실제 환경에서 일반화 가능한 보상 함수와 행동 표현 학습에 대한 관심이 증가하고 있으며, 접촉이 많은 조립 작업 등 복잡한 물리적 상호작용에서의 메타 적응 능력 향상이 중요한 연구 주제로 다뤄지고 있습니다.
