## 📝 상세 요약

📄 *1. Guide, Think, Act: Interactive Embodied Reasoning in Vision-Language-Action Models*

🔗 https://arxiv.org/abs/2605.13632

🏢 *Yiran Ling, Qing Lian, Jinghang Li 외* (소속 기관 명시 안됨)

🔬 *메소드*: 사용자가 어포던스 포인트, 박스, 궤적 등의 시각적 단서로 로봇 정책을 가이드할 수 있는 대화형 VLA 프레임워크를 제안합니다. 외부 가이던스와 내부 작업 계획을 통합하는 통합 공간-시각 Chain-of-Thought를 생성하여 인간의 시각적 의도와 자율적 의사결정을 정렬시킵니다. 실용적 배포를 위해 추론 모듈을 경량화된 반응형 액션 헤드와 결합합니다.

💡 *컨트리뷰션*: 기존 VLA 모델의 직접적인 "감지-행동" 매핑과 달리, 사용자의 공간적 가이던스를 통합하여 시각적 모호성 해결과 실패 복구가 가능한 대화형 추론 메커니즘을 도입했습니다.

🧪 *실험*: SimplerEnv WidowX 벤치마크에서 최고 성능인 81.2% 성공률을 달성했으며, OOD 시각적 변화와 공간적 모호성 상황에서 단일 시각적 상호작용으로 기존 방법 대비 작업 성공률을 크게 개선했습니다.

⭐ **사용자 가이던스를 통해 로봇이 실패 상황에서 복구할 수 있는 대화형 VLA 프레임워크**


📄 *2. RotVLA: Rotational Latent Action for Vision-Language-Action Model*

🔗 https://arxiv.org/abs/2605.13403

🏢 *Qiwei Li, Xicheng Gong, Xinghang Li 외* (소속 기관 명시 안됨)

🔬 *메소드*: 연속적인 회전 잠재 액션 표현을 기반으로 한 VLA 프레임워크를 제안합니다. 잠재 액션을 SO(n) 요소로 모델링하여 연속성, 구성성, 실제 액션 동역학과 정렬된 구조적 기하학을 제공합니다. VLM 백본과 플로우 매칭 액션 헤드로 구성되며, 다운스트림 로봇 제어에서는 잠재 액션이 고수준 가이던스를 제공하는 잠재 플래너 역할을 수행합니다.

💡 *컨트리뷰션*: 기존 LAM의 이산적 양자화 방식 대신 연속적 회전 표현을 도입하여 물리적으로 의미 있는 구조와 향상된 표현 능력을 제공합니다.

🧪 *실험*: 1.7B 파라미터와 1700+ 시간의 사전훈련 데이터로 LIBERO에서 98.2%, RoboTwin2.0에서 깨끗한 환경 89.6%/무작위 환경 88.5% 성능을 달성했으며, 실제 조작 작업에서도 강력한 성능을 보였습니다.

⭐ **연속적 회전 표현을 활용하여 기존 VLA 모델을 능가하는 효율적인 로봇 제어 프레임워크**


📄 *3. Realtime-VLA FLASH: Speculative Inference Framework for Diffusion-based VLAs*

🔗 https://arxiv.org/abs/2605.13778

🏢 *Jiahui Niu, Kefan Gu, Yucheng Zhao 외* (소속 기관 명시 안됨)

🔬 *메소드*: 확산 기반 VLA 모델의 실시간 배포를 위한 투기적 추론 프레임워크를 제안합니다. 경량화된 드래프트 모델과 메인 모델의 Action Expert를 통한 병렬 검증을 도입하고, 필요시 전체 추론 파이프라인으로 되돌아가는 단계별 폴백 메커니즘을 포함합니다. 이를 통해 대부분의 전체 추론 호출을 제거하여 고빈도 재계획을 가능하게 합니다.

💡 *컨트리뷰션*: 확산 기반 VLA의 높은 지연 시간 문제를 해결하여 신뢰성을 희생하지 않으면서 저지연, 고빈도 재계획을 가능하게 했습니다.

🧪 *실험*: LIBERO에서 58.0ms의 전체 추론을 7.8ms의 투기적 라운드로 대체하여 작업 레벨 평균 추론 지연시간을 19.1ms로 단축시켜 3.04배 속도 향상을 달성했으며, 실제 컨베이어 벨트 분류 작업에서도 효과를 입증했습니다.

⭐ **확산 기반 VLA의 추론 속도를 3배 향상시키는 투기적 추론 프레임워크**


📄 *4. FrameSkip: Learning from Fewer but More Informative Frames in VLA Training*

🔗 https://arxiv.org/abs/2605.13757

🏢 *Bin Yu, Shijie Lian, Xiaopeng Lin 외* (소속 기관 명시 안됨)

🔬 *메소드*: VLA 훈련에서 시간적 지도 불균형 문제를 해결하는 데이터 레이어 프레임 선택 프레임워크를 제안합니다. 액션 변화, 시각-액션 일관성, 작업 진행 우선순위, 그리퍼 전환 보존을 사용하여 궤적 프레임을 점수화하고, 목표 유지 비율 하에서 고중요도 프레임으로 훈련 샘플을 재매핑합니다. 데이터로더에서만 작동하여 VLA 아키텍처나 훈련 절차는 변경하지 않습니다.

💡 *컨트리뷰션*: 조작에 중요한 전환 순간은 드물고 변화가 적은 긴 구간이 훈련을 지배하는 시간적 불균형 문제를 식별하고 해결했습니다.

🧪 *실험*: RoboCasa-GR1, SimplerEnv, LIBERO에서 20% 프레임 유지율로 전체 프레임 훈련의 66.50% 대비 76.15%의 매크로 평균 성공률을 달성하여 성공률-유지율 트레이드오프를 개선했습니다.

⭐ **시간적으로 중요한 프레임만 선택하여 VLA 훈련 효율성을 크게 향상시키는 프레임워크**


📄 *5. Learning POMDP World Models from Observations with Language-Model Priors*

🔗 https://arxiv.org/abs/2605.13740

🏢 *Valentin Six, Frederik Panse, Mathis Fajeau 외* (Bernhard Schölkopf 포함)

🔬 *메소드*: 언어 모델 사전 지식을 활용하여 부분 관찰 가능한 환경에서 세계 모델을 학습하는 Pinductor 프레임워크를 제안합니다. LLM이 소수의 관찰-행동 궤적으로부터 후보 POMDP 모델을 제안하고, 신념 기반 우도 점수를 최적화하기 위해 반복적으로 개선합니다. 숨겨진 상태에 대한 특권적 접근 없이도 효과적인 세계 모델 학습을 가능하게 합니다.

💡 *컨트리뷰션*: 기존 방법들이 숨겨진 상태에 대한 특권적 접근을 가정하는 반면, 순수하게 관찰-행동 궤적만으로 POMDP 세계 모델을 효율적으로 학습할 수 있는 방법을 제시했습니다.

🧪 *실험*: 숨겨진 상태에 특권적 접근을 가정하는 LLM 기반 POMDP 학습 방법과 동등한 성능을 달성하면서, 표 형식 POMDP 베이스라인 대비 샘플 효율성을 크게 향상시켰습니다. LLM 능력에 따라 성능이 확장되는 것을 확인했습니다.

⭐ **언어 모델 사전 지식을 활용하여 부분 관찰 환경에서 샘플 효율적인 세계 모델 학습을 실현**

📌 *금주 트렌드 요약*
VLA 모델의 실용성 향상에 초점을 맞춘 연구들이 주를 이루며, 특히 대화형 가이던스, 연속 표현, 추론 최적화, 효율적 훈련 데이터 선택 등을 통해 실시간 배포와 샘플 효율성을 동시에 개선하려는 시도가 돋보입니다. 언어 모델의 사전 지식을 활용한 세계 모델 학습도 새로운 방향으로 주목받고 있습니다.
