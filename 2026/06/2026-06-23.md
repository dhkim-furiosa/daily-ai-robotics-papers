## 📝 상세 요약

📄 *1. SkyJEPA: Learning Long-Horizon World Models for Zero-Shot Sim-to-Real Control of Quadrotors*

🔗 https://arxiv.org/abs/2606.23444

🏢 *Pratyaksh Rao, Wancong Zhang, Randall Balestriero, **Yann LeCun**, Giuseppe Loianno* (NYU)

🔬 *메소드*: JEPA(Joint Embedding Predictive Architecture) 기반의 잠재 공간 동역학 모델을 드론 실시간 제어에 적용한 SkyJEPA를 제안한다. 고정된 잠재 벡터를 해석 가능한 물리적 상태로 변환하는 물리 영감 프로버(physics-inspired prober)를 도입하여 장기 예측의 물리적 타당성을 확보한다. 학습된 모델을 샘플링 기반 최적 제어(MPPI 계열)와 결합하여 임베디드 하드웨어에서 실시간 제어를 수행하며, 자동화된 시뮬레이션 데이터셋 생성 파이프라인으로 실환경 데이터 의존성을 최소화한다.

💡 *컨트리뷰션*: 기존 JEPA 방식이 주로 저주파 운동학적 계획에 국한되었던 반면, SkyJEPA는 고주파 제어가 필요한 쿼드로터에 적용하여 제로샷 Sim-to-Real 전이를 성공적으로 달성한 최초의 사례이다.

🧪 *실험*: 야외 폐루프 비행 실험 및 오픈루프 예측 실험에서 평가하였으며, 시뮬레이션만으로 학습한 모델이 실환경에서 강건한 제어 성능과 다양한 운용 조건에 대한 일반화를 입증하였다 (구체적 수치는 논문 내 상세 기재).

⭐ **Yann LeCun의 JEPA 철학을 드론 고주파 실시간 제어에 적용하여, 시뮬레이션 데이터만으로 학습한 모델의 제로샷 실환경 전이를 최초로 실현한 연구**



📄 *2. SC3-Eval: Evaluating Robot Foundation Models via Self-Consistent Video Generation*

🔗 https://arxiv.org/abs/2606.18610

🏢 *Wei-Cheng Tseng, Gashon Hussein, Yuzhu Dong, Allen Z. Ren, Lucy X. Shi, XuDong Wang, **Sergey Levine**, Zhaoshuo Li, Jinwei Gu, Florian Shkurti, Ming-Yu Liu, Quan Vuong* (UC Berkeley 등)

🔬 *메소드*: 사전 학습된 비디오 파운데이션 모델을 정책 평가기로 변환하는 SC3-Eval을 제안하며, 세 가지 일관성 조건을 부과한다. (1) 순방향-역방향 동역학 일관성: 액션으로 프레임을 예측하고 프레임으로 액션을 복원하는 공동 학습으로 드리프트를 억제한다. (2) 크로스뷰 일관성: 카메라 뷰 간 인페인팅으로 멀티뷰 관측의 장기적 정합성을 유지한다. (3) 테스트 타임 일관성: 역방향 동역학 모드를 불확실성 신호로 활용하여 롤아웃을 조기 종료한다.

💡 *컨트리뷰션*: 기존 비디오 모델 기반 평가 방식이 단순 집계 순위에 그쳤던 것과 달리, SC3-Eval은 실제 실환경 실패 모드를 재현하여 세밀한 진단적 비교를 가능하게 한다.

🧪 *실험*: 7가지 실환경 VLA(Vision-Language-Action) 정책을 대상으로 평가하였으며, 폐루프 Pearson 상관계수 **0.929**, MMRV **0.119**를 달성하여 3개의 강력한 베이스라인을 상회하고 신규 태스크에도 일반화하였다.

⭐ ***Sergey Levine** 연구팀이 제안한 SC3-Eval은 세 가지 자기일관성 조건으로 비디오 생성 모델을 정책 평가기로 전환하여, 실환경 없이도 VLA 정책을 0.929의 높은 상관관계로 평가하는 확장 가능한 프레임워크**



📄 *3. Learning Process Rewards via Success Visitation Matching for Efficient RL*

🔗 https://arxiv.org/abs/2606.23640

🏢 *Raymond Tsao, Andrew Wagenmaker, **Sergey Levine*** (UC Berkeley)

🔬 *메소드*: 희소 보상(sparse reward)을 밀집 과정 보상(dense process reward)으로 변환하는 SVM(Success Visitation Matching) 방식을 제안한다. 과거의 성공/실패 에피소드를 구별하도록 판별기(discriminator)를 학습시키고, 이 판별기를 보상으로 활용하여 RL 정책이 성공 에피소드의 상태-액션 방문 분포를 모방하도록 유도한다. 이론적으로 최적 정책을 변경하지 않으면서 밀집 피드백을 제공함을 증명한다.

💡 *컨트리뷰션*: 기존 희소 보상 문제에 대한 접근법들과 달리, 성공 에피소드 전체의 방문 분포를 매칭 목표로 삼아 크레딧 할당 문제를 해결하면서도 최적성을 이론적으로 보장한다.

🧪 *실험*: 시뮬레이션 및 실환경 로봇 조작 태스크에서 파인튜닝 실험을 수행하였으며, 희소 보상만 사용하는 기준선 대비 RL 파인튜닝 수렴 속도에서 유의미하게 빠른 성능 향상을 달성하였다.

⭐ ***Sergey Levine** 팀이 판별기 기반 방문 분포 매칭으로 희소 보상을 밀집 보상으로 자동 변환하여, 로봇 조작 RL 파인튜닝 효율을 이론적 최적성 보장 하에 대폭 향상시킨 연구**



📄 *4. You Don't Need Strong Assumptions: Visual Representation Learning via Temporal Differences*

🔗 https://arxiv.org/abs/2606.15956

🏢 *Ninad Daithankar, Alexi Gladstone, **Yann LeCun**, Heng Ji* (NYU / UIUC)

🔬 *메소드*: 비디오로부터의 자기지도 학습 패러다임인 TDV(Temporal Difference in Vision)를 제안한다. 이미지 인코더와 모션 인코더를 공동 학습하되, "현재 프레임 표현 + 인코딩된 모션 = 다음 프레임 표현"이라는 시간적 인과 관계만을 귀납적 편향으로 사용한다. 증강(augmentation), 마스킹, 크롭핑 등 기존 SSL의 강한 귀납적 편향을 일체 사용하지 않는다.

💡 *컨트리뷰션*: 데이터가 증가할수록 강한 귀납적 편향이 오히려 병목이 됨을 실험으로 확인하고, 인과적 시간 가정만으로 기존 최신 SSL 방법과 동등한 성능을 달성하는 미니멀리스트 표현 학습 패러다임을 제시한다.

🧪 *실험*: 밀집 공간 태스크(dense spatial tasks) 벤치마크에서 평가하였으며, 증강·마스킹 등 강한 귀납적 편향 없이도 SOTA SSL 레시피와 동등한 성능을 기록하였고, 데이터 규모 증가 시 귀납적 편향의 최적 강도가 감소함을 정량적으로 확인하였다.

⭐ ***Yann LeCun** 연구팀이 제안한 TDV는 증강·마스킹 없이 시간적 인과성만으로 비디오 자기지도 표현 학습을 수행하며, 데이터 스케일 시대에 강한 가정이 오히려 병목임을 증명한 패러다임 전환 연구**



📄 *5. SPIRAL: Learning to Search and Aggregate*

🔗 https://arxiv.org/abs/2606.23595

🏢 *Jubayer Ibn Hamid, Ifdita Hasan Orney, Michael Y. Li, Omar Shaikh, Yoonho Lee, Dorsa Sadigh, **Chelsea Finn**, Noah Goodman* (Stanford University)

🔬 *메소드*: 순차(sequential), 병렬(parallel), 집계(aggregative) 세 가지 추론 프리미티브를 통합하는 SPIRAL(Sequential-Parallel-Aggregative Reinforcement Learning) 프레임워크를 제안한다. 언어 모델이 먼저 병렬로 다수의 독립적인 CoT 트레이스를 샘플링한 뒤, 이를 조건으로 최종 집계 트레이스를 생성하며, 전체 구성 요소를 최종 집계 응답의 보상에 대해 엔드투엔드로 최적화한다. 집합 강화학습(set RL)으로 집합적으로 유용한 트레이스 생성을 학습하고, 표준 RL로 집계 능력을 학습한다.

💡 *컨트리뷰션*: 기존 포스트 트레이닝이 단일 트레이스 내 순차 추론만 최적화했던 것과 달리, SPIRAL은 테스트 타임 컴퓨트 스케일링의 세 가지 프리미티브 모두를 학습 단계에서 통합적으로 최적화한다.

🧪 *실험*: 다양한 추론 벤치마크에서 평가하였으며, GRPO 대비 추론 컴퓨트 스케일링 효율 최대 **11배** 향상, 성능 **15% 향상**을 달성하였다.

⭐ ***Chelsea Finn** 연구팀의 SPIRAL은 순차·병렬·집계 추론을 RL로 통합 최적화하여 GRPO 대비 11배 스케일링 효율과 15% 성능 향상을 달성한 테스트 타임 컴퓨트 최적화 프레임워크**


📌 *금주 트렌드 요약*

이번 주 논문들은 공통적으로 **귀납적 편향(inductive bias)을 최소화하면서도 확장성과 효율성을 극대화**하는 방향으로 수렴한다. 로봇 제어(SkyJEPA, SC3-Eval, SVM), 표현 학습(TDV), 언어 모델 추론(SPIRAL) 전 영역에서 **기존의 강한 가정을 데이터·컴퓨트 기반의 자기일관성 또는 인과적 구조로 대체**하려는 시도가 두드러진다. 특히 Yann LeCun과 Sergey Levine 그룹을 중심으로, **희소 신호(희소 보상, 인과적 시간 관계)로부터 밀집하고 풍부한 학습 신호를 이끌어내는 방법론**이 AI·로보틱스 연구의 핵심 트렌드로 자리잡고 있다.
