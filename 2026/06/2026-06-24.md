📄 *1. G³VLA: Geometric Inductive Bias for Vision-Language-Action Models*

🔗 https://arxiv.org/abs/2606.24472

🏢 *기관/저자*: Yue Peng, Yongzhe Zhao, Artur Habuda, Khuyen Pham, Yanheng Zhu, Tran Nguyen Le, Fares Abu-Dakka, Li Guo (소속 기관 미상)

🔬 *메소드*: 기존 VLA 모델의 시각 토큰이 2D 이미지 좌표에만 머무는 한계를 극복하기 위해, 카메라 내/외부 파라미터를 활용한 기하학적 정보를 VLA의 시각 토큰 스트림에 주입하는 G³VLA 모듈을 제안한다. 구체적으로 intrinsic 조건부 ray embedding, 투영 위치 인코딩(PRoPE), 양방향 교차 뷰 융합(bidirectional cross-view fusion)의 세 가지 요소를 결합한다. 깊이 센서나 수동 어노테이션 없이, 사용 가능할 경우 ground-truth 포인트 맵을, 아닐 경우 confidence-gated π³X 교사 예측을 기하학적 지도 신호로 활용한다.

💡 *컨트리뷰션*: 기존 VLA 모델의 액션 공간이나 모방 학습 목표를 변경하지 않으면서도, 멀티 카메라 환경에서 교정된 기하학 구조를 시각 토큰에 직접 통합하는 플러그인 방식의 카메라 인식 기하학 모듈을 최초로 제안했다.

🧪 *실험*: LIBERO 스위트, RoboCasa24, RoboTwin2.0, 실제 로봇 환경에서 검증하였으며, π₀에 적용 시 공간적·객체 민감 태스크에서 가장 큰 성능 향상을 기록했다. π₀.₅ 및 GR00T 1.5에서도 추가 검증하였고, 기하학 인식 토큰이 액션 생성 경로에 직접 접근할 수 있을 때 기하학적 전이 효과가 가장 크다는 결과를 확인했다. (구체적 수치는 논문 본문 및 프로젝트 페이지 참조)

⭐ **VLA 모델에 카메라 캘리브레이션 기반 기하학적 귀납 편향을 주입하여, 멀티 카메라 로봇 조작에서 공간 인식 능력을 크게 향상시킨다.**



📄 *2. Data Standards for Humanoid Robotics: The Missing Infrastructure for Physical AI*

🔗 https://arxiv.org/abs/2606.19769

🏢 *기관/저자*: Shaoshan Liu, Xiugong Qin, Xuan Wu, Xuan Xia, Ning Ding, Jialu Liu, Jie Tang (ISO/TC 299/WG 16, ISO/WD 26264-1 표준 개발 참여 저자들)

🔬 *메소드*: 휴머노이드 로봇 데이터를 단순한 디지털 샘플 모음이 아닌 "체화된 상호작용 데이터"로 정의하고, ISO/WD 26264-1 표준 개발 경험을 바탕으로 Physical AI를 위한 데이터 표준의 필요성과 설계 원칙을 체계적으로 논증한다. 타이밍, 좌표 프레임, 캘리브레이션, 기구학, 단위, 동기화 정보의 물리적 일관성(physical coherence)을 보존해야 멀티모달 스트림이 재사용 가능함을 주장한다. 수평적 인프라(생명주기 관리, 메타데이터, 출처, 품질, 버전 관리)와 능력별 수직 도메인 문법(조작, 이동, HRI, 인지 등)으로 구성되는 이중 구조의 표준 프레임워크를 제안한다.

💡 *컨트리뷰션*: 휴머노이드 로봇 확장성의 병목이 데이터 부족이 아니라 높은 수집 비용, 데이터 사일로, 비일관적 평가로 인한 "비누적적(non-cumulative) 데이터" 문제임을 지적하고, 이를 해결할 국제 표준의 필요성을 처음으로 체계적으로 논증했다.

🧪 *실험*: 이 논문은 실험 기반 논문이 아닌 포지션/표준 제안 논문으로, ISO/WD 26264-1 표준화 작업에서 도출된 실제 사례 분석과 현존 로봇 데이터셋의 한계를 근거로 논증을 전개한다.

⭐ **휴머노이드 로봇의 Physical AI 확장을 가로막는 진짜 병목은 데이터 표준의 부재이며, ISO 기반 국제 표준이 체화된 경험의 축적·공유·재사용을 가능하게 하는 핵심 인프라가 될 것이다.**



📄 *3. LaGO: Latent Action Guidance for Online Reinforcement Learning*

🔗 https://arxiv.org/abs/2606.24669

🏢 *기관/저자*: Kuan-Yen Liu, Ren-Jyun Huang, Ti-Rong Wu (소속 기관 미상)

🔬 *메소드*: 사전 학습된 LLM을 명시적 플래너나 컨트롤러로 사용하는 대신, LLM의 출력을 잠재 액션 사전(latent action prior)으로 활용하여 온라인 정책 최적화를 소프트하게 유도하는 LaGO 프레임워크를 제안한다. LLM이 생성하는 잠재 표현이 PPO 기반 온라인 강화학습의 탐색 방향을 안내함으로써, 직접 액션 생성의 부정확성 문제를 우회한다. LLM의 자연어 추론 능력을 정책 학습의 귀납 편향으로 전환하는 방식으로, 정밀한 액션 생성 없이도 LLM의 계획 지식을 활용한다.

💡 *컨트리뷰션*: LLM을 직접 컨트롤러로 쓰는 기존 방식의 신뢰성 문제를 해결하기 위해, LLM을 잠재 공간에서의 소프트 가이던스 신호원으로 재정의하는 새로운 패러다임을 제시했다.

🧪 *실험*: 이산 제어 벤치마크 CLEVR-Robot과 연속 제어 벤치마크 Meta-World에서 실험하였으며, CLEVR-Robot에서 Vanilla PPO 대비 평균 성공률을 15.1% → 27.2%로, Meta-World에서는 2.7% → 15.2%로 향상시켰다. 더 강력한 사전 학습 LLM일수록 더 효과적인 가이던스를 제공함을 추가 분석으로 확인했다.

⭐ **LLM을 직접 컨트롤러 대신 잠재 액션 사전으로 활용함으로써, 온라인 강화학습의 탐색 효율을 크게 높이고 다양한 제어 벤치마크에서 일관된 성능 향상을 달성한다.**



📄 *4. ENPIRE: Agentic Robot Policy Self-Improvement in the Real World*

🔗 https://arxiv.org/abs/2606.19980

🏢 *기관/저자*: Wenli Xiao, Jia Xie, Tonghe Zhang, Haotian Lin, Letian "Max" Fu, Haoru Xue, Jalen Lu, Yi Yang, Cunxi Dai, Zi Wang, Jimmy Wu, Guanzhi Wang, S. Shankar Sastry, Ken Goldberg, Linxi "Jim" Fan, Yuke Zhu, Guanya Shi (UC Berkeley, NVIDIA, Carnegie Mellon University 등)

🔬 *메소드*: 코딩 에이전트가 실제 로봇 환경에서 자율적으로 정책을 개선할 수 있도록, 물리적 피드백 루프를 자동화하는 ENPIRE 하네스 프레임워크를 제안한다. EN(환경 자동 리셋·검증), PI(정책 개선 실행), R(단일/복수 로봇 롤아웃 평가), E(코딩 에이전트의 로그 분석·문헌 참조·코드 개선) 네 개의 핵심 모듈로 구성된 폐루프 시스템을 구성한다. 이 시스템은 장면 리셋 → 정책 실행 → 결과 검증 → 다음 반복 개선의 반복 가능한 물리적 피드백 루틴을 자동화하여, 인간 개입을 최소화한다.

💡 *컨트리뷰션*: 코딩 에이전트의 성공이 디지털 환경에 국한되던 한계를 극복하여, 실제 물리 세계에서 반복 가능한 자율적 로봇 정책 개선 루프를 최초로 구현한 프레임워크를 제시했다.

🧪 *실험*: 핀 박스 정리, 집타이 조이기, 도구 사용 등 고난도 정밀 조작 태스크에서 프런티어 코딩 에이전트가 자율적으로 **99% 성공률**의 정책을 학습하는 데 성공하였으며, 에이전트 팀을 로봇 플릿에 병렬 배치 시 학습 속도가 추가로 가속되었다.

⭐ **코딩 에이전트 + 자동화된 물리적 피드백 루프의 결합으로, 인간 감독 없이 실제 로봇이 99% 성공률의 정밀 조작 정책을 스스로 학습하는 새로운 자율 로보틱스 패러다임을 제시한다.**



📄 *5. InSight: Self-Guided Skill Acquisition via Steerable VLAs*

🔗 https://arxiv.org/abs/2606.24884

🏢 *기관/저자*: Maggie Wang, Lars Osterberg, Stephen Tian, Ola Shorinwa, Jiajun Wu, Mac Schwager (Stanford University)

🔬 *메소드*: VLA 모델을 프리미티브 액션 수준("그리퍼를 그릇으로 이동", "위로 들어올리기" 등)에서 조향 가능(steerable)하게 만들어 자율적 스킬 획득을 가능하게 하는 InSight 프레임워크를 제안한다. 첫 번째 단계에서는 VLM 계획 분해와 엔드이펙터 포즈를 활용해 데모를 레이블된 프리미티브로 자동 분할하는 파이프라인을 구축하고, 두 번째 단계에서는 VLM이 새로운 태스크에 필요한 누락 프리미티브를 식별·시도·레이블링·통합하는 데이터 플라이휠을 구동한다. 성공한 데모는 자동으로 VLA 학습 데이터셋에 통합되어 지속적인 스킬 확장이 이루어진다.

💡 *컨트리뷰션*: 인간 데모 없이도 VLA가 새로운 프리미티브 스킬을 자율적으로 습득하고, 이를 조합하여 장기 복합 태스크를 수행할 수 있는 지속적 스킬 획득 프레임워크를 제시했다.

🧪 *실험*: 시뮬레이션 및 실제 환경에서 블록 뒤집기, 서랍 닫기, 쓸기, 비틀기, 붓기 등 5개 태스크에 대해 해당 스킬의 인간 데모 없이 검증하였다. 학습된 프리미티브들은 추가 데모 없이 조합되어 새로운 장기 태스크를 수행할 수 있음을 확인했다. (태스크별 구체적 성공률은 프로젝트 웹사이트 참조)

⭐ **VLA를 프리미티브 수준에서 조향 가능하게 만들고 VLM 기반 데이터 플라이휠을 결합함으로써, 인간 데모 없이
