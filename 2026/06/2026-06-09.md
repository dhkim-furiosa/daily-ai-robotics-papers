## 📝 상세 요약

📄 *1. Robotic Policy Adaptation via Weight-Space Meta-Learning*

🔗 https://arxiv.org/abs/2606.07217

🏢 *Christian Bianchi, Siamak Yousefi, Alessio Sampieri, Andrea Roberti, Luca Rigazio, Fabio Galasso, Luca Franco*

🔬 *메소드*: WIZARD는 언어 지시와 짧은 데모 비디오만으로 VLA 정책에 대한 태스크별 LoRA 파라미터를 생성하는 weight-space 메타 러닝 프레임워크입니다. 메타 트레이닝 중에 태스크 증거를 전문가 LoRA 업데이트로 직접 매핑하는 방법을 학습하여, 태스크별 액션 라벨이나 테스트 타임 최적화 없이 단일 forward pass로 적응 가중치를 예측합니다.

💡 *컨트리뷰션*: 기존 VLA 모델들이 새 태스크 적응 시 필요했던 태스크별 데모, 액션 어노테이션, 추가 파인튜닝을 완전히 우회하는 방법을 제안했습니다.

🧪 *실험*: LIBERO에서 미본 데이터셋 컬렉션에 대해 최대 2배, 미본 태스크에 대해 최대 14배 성능 향상을 달성했으며, Franka Emika Panda 실제 로봇에서도 일관된 성능 개선을 보였습니다.

⭐ **언어 지시와 짧은 데모만으로 VLA 정책을 즉시 적응시키는 weight-space 메타 러닝 프레임워크**


📄 *2. Simulation-Driven Imitation Learning for Biosignals-Free Shared-Autonomy Prosthetic Grasping*

🔗 https://arxiv.org/abs/2606.07389

🏢 *Kaijie Shi, Wanglong Lu, Huiling Chen, Vinicius Prado da Fonseca, Ting Zou, Hanli Zhao, Xianta Jiang*

🔬 *메소드*: 손목 장착 가상 카메라에서 다양한 reach-to-grasp 데모를 자동 생성하는 확장 가능한 시뮬레이션 프레임워크를 제안합니다. 물리적으로 실현 가능한 파지 합성, 자연스러운 도달 궤적 리타겟팅, reach-grasp-lift 실행을 절차적으로 생성된 실내 환경에서 결합하여 대규모 데모 데이터셋을 구축합니다.

💡 *컨트리뷰션*: EMG 신호 없이도 의수 제어가 가능한 생체신호 무관 공유 자율성 접근법으로, 실제 인간 데모 데이터 수집의 비용과 가변성 문제를 시뮬레이션으로 해결했습니다.

🧪 *실험*: 세 가지 현실적 설정에서 학습된 sim-to-real 정책이 90% 이상의 파지 성공률을 달성했으며, 베이스라인 방법들을 능가하고 더 강한 일반화 능력을 보였습니다.

⭐ **시뮬레이션 기반 대규모 데모 데이터로 EMG 신호 없이도 90% 이상 파지 성공률을 달성하는 의수 제어 시스템**


📄 *3. Supervision versus Demonstration-Based In-Context Learning for Multiword Expression Classification*

🔗 https://arxiv.org/abs/2606.07479

🏢 *Sercan Karakaş, Yusuf Şimşek*

🔬 *메소드*: 터키어 관용적 경동사 구문(LVC) 검출을 이진 분류 태스크로 정의하고, 수동으로 생성된 제어 세트에서 지도학습 터키어 인코더 베이스라인과 zero-shot, one-shot, few-shot 프롬프팅을 적용한 세 가지 instruction-tuned LLM을 비교합니다. 데모가 오류 프로필에 미치는 영향을 분석하여 터키어 메타언어학적 분류에서의 프롬프트 민감도를 조사합니다.

💡 *컨트리뷰션*: 터키어 다어구 표현 처리에서 LLM의 프롬프트 민감도를 체계적으로 분석하고, 신중하게 구성된 데모가 LVC 검출 성능을 크게 개선할 수 있음을 보였습니다.

🧪 *실험*: 147개 샘플의 제어 세트에서 GPT-OSS-20B와 Qwen 2.5-14B가 few-shot 프롬프팅으로 지도학습 베이스라인과 비슷하거나 더 나은 성능을 달성했습니다.

⭐ **터키어 관용구 검출에서 LLM의 프롬프트 민감도를 분석하고 few-shot 프롬프팅의 효과를 입증**


📄 *4. Skill-3D: Evolving Scene-Aware Skills for Agentic 3D Spatial Reasoning*

🔗 https://arxiv.org/abs/2606.07436

🏢 *Haoyuan Li, Zhengdong Hu, Jun Wang, Hehe Fan, Yi Yang*

🔬 *메소드*: 3D 공간 추론 태스크의 장면별 이질성을 해결하기 위해 자기진화하는 장면 인식 스킬을 학습하는 Skill-3D 프레임워크를 제안합니다. 태스크 장면을 식별하고 에이전트의 도구 사용 궤적을 Scene Memory에 기록하여, 유사한 장면의 성공적인 궤적들을 집계하고 재사용 가능한 장면 인식 스킬로 증류합니다.

💡 *컨트리뷰션*: 기존 방법들이 모든 장면에 동일한 도구 사용 전략을 적용하는 문제를 해결하여, 장면과 태스크에 따른 적응적 도구 선택을 가능하게 했습니다.

🧪 *실험*: VSI-Bench에서 도구 활용률을 39%에서 78%로 크게 향상시켰으며, MMSI-Bench에서 Gemini-3-Flash를 67% 개선시켰고, VSI-Bench에서 Qwen3-VL-8B를 43% 향상시켰습니다.

⭐ **장면별 맞춤형 도구 사용 전략을 학습하여 3D 공간 추론에서 도구 활용률을 39%에서 78%로 대폭 향상**


📄 *5. RhinoVLA Technical Report*

🔗 https://arxiv.org/abs/2606.07383

🏢 *Huixi Intelligence: Chen Zhang, Chenyang Zhou, Guanglei Ding, Guanghui He, Haibin Gao, Jiajia Chen, Jianyong Zhang, Lianyi Yu, Ningyi Xu, Ping Xu, Qingchen Li, Yingjun Hu, Yijia Zhang, Yuxi Liu* (Huixi Intelligence 특별 강조)

🔬 *메소드*: VLM 시각 및 컨텍스트 토큰이 배포 지연시간의 주요 원인임을 파악하고, 토큰 효율적인 Qwen3-VL 백본과 연속 Action Expert를 채택한 배포 지향적 VLA 모델을 제안합니다. View Registry, 72D 물리적 상태-액션 슬롯 공간, robot-instance LoRA를 결합한 통합 인터페이스로 이종 로봇 간 학습을 지원합니다.

💡 *컨트리뷰션*: 하드웨어 인식 컴파일, 혼합 정밀도 실행, 병렬 시각 인코딩을 통한 엣지 하드웨어 최적화로 실시간 로봇 제어를 가능하게 했습니다.

🧪 *실험*: π0.5와 유사한 파라미터 스케일에서 비교 가능한 다운스트림 성능을 달성하면서, Huixi R1에서 11.69 Hz 종단간 추론으로 10 Hz 실시간 폐루프 제어 목표를 달성했습니다.

⭐ **엣지 하드웨어에서 11.69 Hz 실시간 추론이 가능한 배포 최적화된 VLA 모델로 실용적 로봇 제어 실현**

📌 *금주 트렌드 요약*
VLA 모델의 실용화가 가속화되고 있으며, 메타 러닝을 통한 빠른 적응(WIZARD)과 엣지 하드웨어 배포 최적화(RhinoVLA)가 핵심 트렌드입니다. 시뮬레이션 기반 대규모 데이터 생성이 실제 환경의 데이터 부족 문제를 해결하는 주요 방법론으로 부상하고 있으며, 장면 인식 스킬 학습을 통한 적응적 도구 사용이 3D 공간 추론의 새로운 패러다임을 제시하고 있습니다.
