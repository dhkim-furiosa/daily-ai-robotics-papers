## 📝 상세 요약

📄 *1. Patch Policy: Efficient Embodied Control via Dense Visual Representations*

🔗 https://arxiv.org/abs/2607.18236

🏢 *NYU (Gaoyue Zhou, Zichen Jeff Cui, Ada Langford, Bowen Tan, Lerrel Pinto)* — ⭐ **Yann LeCun** 공동 저자 (Meta / NYU)

🔬 *메소드*: Vision Transformer(ViT)의 사전학습된 밀집 패치 토큰(dense patch tokens)을 전체 VLM 없이 직접 정책 모델에 입력할 수 있도록 하는 경량 아키텍처 확장인 Patch Policy를 제안한다. 핵심은 블록-인과적 어텐션 마스크(block-causal attention mask)로, 표준 정책의 시간적 인과성을 유지하면서 관측당 다수의 패치 토큰과 상태 정보를 동시에 처리할 수 있게 한다. 이를 통해 글로벌 풀링 없이 공간적 세부 정보를 보존하면서도 추론 속도와 학습 효율을 유지한다.

💡 *컨트리뷰션*: 기존 VLA 모델들이 수십억 파라미터의 VLM 백본을 요구했던 것과 달리, OpenVLA-OFT 대비 약 0.7%의 파라미터만으로 밀집 패치 피처를 활용할 수 있는 경량 파이프라인을 최초로 제시한다.

🧪 *실험*: 4개의 시뮬레이션 환경과 3개의 실제 로봇 환경에서 평가했으며, 최신 글로벌 풀링 표현 대비 40% 상대적 성능 향상을 달성했다. 파인튜닝된 OpenVLA-OFT 대비 18% 성능 향상을 보였으며, 사용된 파라미터는 약 0.7% 수준에 불과하다.

⭐ **ViT의 밀집 패치 토큰을 블록-인과 어텐션으로 경량 활용하여, 거대 VLM 없이도 로봇 제어 성능을 크게 향상시킨 효율적 정책 아키텍처**



📄 *2. PhyAgentOS: A Self-Evolving Operating System for Embodied Agents with Decoupled Cognitive Planning and Physical Execution*

🔗 https://arxiv.org/abs/2607.16636

🏢 *Sun Yat-sen University (Yang Liu, Weixing Chen 외, Liang Lin)*

🔬 *메소드*: VLA 모델, 월드 모델, 에이전트 플래너를 통합하는 런타임 운영체제인 PhyAgentOS를 제안하며, 세션(session)을 스케줄링의 최소 단위로 삼는 Session-Centered Runtime을 채택한다. 인지(cognition)와 물리적 실행(physical execution)을 분리하기 위해 State-as-a-File 프로토콜을 도입하여 크로스-레이어 상태를 Markdown+YAML 파일로 구체화함으로써 코드 의존성 없이 검사·버전 관리가 가능하다. SessionVerifier가 실행 종료와 의미론적 태스크 완료를 구분하고, 검증된 결과를 에피스템 메모리(epistemic memory)로 통합해 재학습 없이 지속적 지식 축적이 이루어진다.

💡 *컨트리뷰션*: 이기종 구현체(heterogeneous embodiments) 간에 공통 실행 추상화, 공유 상태, 의미론적 검증, 지속적 경험을 제공하는 최초의 체화 에이전트 전용 운영체제 수준 런타임을 제시한다.

🧪 *실험*: Optimus-67, StarDojo, DST-Dojo 벤치마크에서 평가하고, 19개 이상의 시뮬레이션·실물 구현체에서 검증했다. LIBERO, Calvin, RoboCasa365에서 다수의 VLA 모델에 걸쳐 성능 향상을 달성했으며, 게임→시뮬레이션→실물 로봇의 점진적 검증 전략을 통해 인지 레이어의 일관성을 확인했다.

⭐ **체화 에이전트를 위한 OS 수준의 런타임을 제안하여, 인지와 물리 실행을 파일 시스템으로 분리하고 재학습 없이 자기진화하는 에이전트 인프라를 구현**



📄 *3. Apple-π: Benchmarking Thinking with Video Towards Law-Grounded Physical Intelligence*

🔗 https://arxiv.org/abs/2607.16401

🏢 *Nanyang Technological University, S-Lab (Runmao Yao, Ziwei Liu 외)*

🔬 *메소드*: 물리 법칙에 명시적으로 기반한 비디오 모델 평가 벤치마크 Apple-PI를 제안하며, 고전역학의 10가지 표준 태스크를 포함하는 400개 비디오 데이터셋 Orchard를 구축한다. 과학적 추론에 기반한 3단계 프로토콜(Perception → Formulation → Deduction)을 설계하고, 인포그래픽이 주석된 첫 프레임에 chain-of-frames 프롬프팅을 적용하여 생성된 비디오를 모델의 추론 흔적으로 간주한다. MLLM 기반 주관적 채점과 물리 법칙 기반 객관적 측정을 결합한 하이브리드 평가 스위트로 단계별 오류 진단을 가능하게 한다.

💡 *컨트리뷰션*: 기존 벤치마크가 출력 수준의 물리적 그럴듯함만 평가했던 것과 달리, 모델이 법칙에 근거한 추론 과정을 충실히 거쳤는지를 단계별로 진단하는 최초의 물리 법칙 기반 비디오 평가 프레임워크를 제시한다.

🧪 *실험*: 11개의 비디오 생성 모델을 평가한 결과, 최고 성능 모델도 0.473점에 불과하여 현재 모델들이 신뢰할 수 있는 물리 법칙 기반 월드 시뮬레이터와 거리가 멀다는 것을 확인했다. Perception→Formulation→Deduction 병목 현상, 다중 법칙 상태 전이 취약성, Sim-to-Real 격차가 주요 분석 결과로 도출되었다.

⭐ **비디오 생성 모델이 진정한 물리 법칙 기반 월드 모델인지를 3단계 추론 과정으로 진단하는 최초의 법칙-기반 벤치마크로, 현존 최고 모델도 0.473점에 불과함을 폭로**



📄 *4. FM-VLA: Force-based Memory for Vision-Language-Action Models in Contact-Rich Manipulation*

🔗 https://arxiv.org/abs/2607.18231

🏢 *Microsoft Research Asia (Ruicheng Li, Qixiu Li 외, Jiaolong Yang, Baining Guo)*

🔬 *메소드*: 접촉이 많은 조작 태스크에서의 비-마르코프적(non-Markovian) 시간 추론을 위해 힘(force) 이력을 기억으로 활용하는 FM-VLA를 제안한다. 힘 시계열 복원으로 사전학습된 VAE(Variational Autoencoder)를 통해 힘 이력을 컴팩트한 포스 메모리 토큰으로 인코딩하고, 이를 짧은 상태 이력과 함께 액션 전문가 모듈의 추가 조건부 토큰으로 투영한다. 이를 통해 VLA 모델이 누적된 접촉 이벤트 이력을 활용해 조작을 유도할 수 있게 된다.

💡 *컨트리뷰션*: 기존 메모리 증강 VLA가 과거 이미지나 언어 요약에 의존하여 시각적으로 모호한 접촉 이벤트를 처리하지 못했던 한계를 극복하고, 힘 센서 데이터를 경량 VAE 토큰으로 압축해 최소한의 추론 오버헤드로 접촉 이력 추론을 가능하게 했다.

🧪 *실험*: 숨겨진 블록 찾기, 버튼 누르기, 특정 횟수만큼 접시 닦기 등 메모리 의존적 3가지 태스크에서 평가했으며, 경량 포스 메모리를 적용한 FM-VLA가 80% 이상의 성공률을 달성하여 기존 베이스라인 대비 유의미한 성능 향상을 보였다.

⭐ **VAE로 압축된 힘 이력 토큰을 VLA에 주입하여 시각적으로 구별하기 어려운 반복적 접촉 태스크에서 80% 이상 성공률을 달성한 포스-메모리 증강 VLA**



📄 *5. RynnBrain 1.1: Towards More Capable and Generalizable Embodied Foundation Model*

🔗 https://arxiv.org/abs/2607.17977

🏢 *Rynn Robotics / Nanyang Technological University (Kehan Li, Bohan Hou 외, Shijian Lu, Jianfei Yang, Deli Zhao)*

🔬 *메소드*: 2B, 9B, 122B-A10B 세 가지 스케일을 아우르는 체화 파운데이션 모델 패밀리 RynnBrain 1.1을 제안하며, 통합 시공간·물리 기반(spatio-temporal and physically grounded) 프레임워크로 학습된다. 이전 버전 대비 전 모델 패밀리에 접촉점 예측(contact-point prediction)을 추가하고, 2B·9B 모델에는 네이티브 3D 그라운딩을 도입하여 로봇 조작에 직접 정렬된 표현을 생성한다. 통합 크로스-에봄디먼트 액션 공간과 구현체별 마스킹을 적용한 RynnBrain-VLA를 Unitree G1, Astribot-S1, Tianji-Wuji에 배포했다.

💡 *컨트리뷰션*: 단일 파운데이션 모델에서 체화 인지·로컬리제이션·3D 그라운딩·조작까지 통합하고, 멀티태스크·멀티-에봄디먼트 공동 학습이 개별 태스크 학습 대비 프로세스 점수와 성공률 모두를 향상시킴을 실증한다.

🧪 *실험*: VSI-Bench, MMSI, RefSpatial-Bench에서 122B-A10B 모델이 평가된 모든 독점·오픈소스 모델을 능가했다. 실물 로봇 실험에서 RynnBrain 초기화 정책이 Qwen 기반 및 대표적인 제너럴리스트 VLA를 상회했으며, 멀티태스크·멀티-에봄디먼트 공동 학습이 단일 태스크 학습 대비 일관된 성능 향상을 보였다.

⭐ **2B~122B 스케일의 체화 파운데이션 모델 패밀리로 접촉점 예측과 3D 그라운딩을 통합하여 공간 추론 벤치마크에서 모든 기존 모델을 능가**


📌 *금주 트렌드 요약*

이번 주 논문들은 공통적으로 **VLA(Vision-Language-Action) 모델의 한계를 다양한 방향에서 확장**하는 흐름을 보인다. 경량 밀집 표현(Patch Policy), 힘 센서 기반 메모리(FM-VLA), 대규모 파운데이션 모델(RynnBrain 1.1)은 각각 효율성·접촉 인식·스케일 측면에서 VLA를 고도화하고, PhyAgentOS는 이를 시스템 수준에서 통합·관리하는 OS 패러다임을 제시한다. 또한 Apple-PI 벤치마크는 현재 비디오/월드 모델이 물리 법칙을 진정으로 이해하지 못한다는
