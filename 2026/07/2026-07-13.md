## 📝 상세 요약

📄 *1. Seeing is Free, Speaking is Not: Uncovering the True Energy Bottleneck in Edge VLM Inference*

🔗 https://arxiv.org/abs/2607.09520

🏢 *기관/저자*: Junfei Zhan, Haoxun Shen, Mingang Guo, Zixuan Huang, Tengjiao He

🔬 *메소드*: 엣지 하드웨어(NVIDIA RTX 3070, Jetson Orin NX)에서 5개 VLM 모델, 3개 아키텍처 계열, 4가지 입력 해상도에 걸쳐 최초의 체계적인 에너지 프로파일링을 수행했습니다. 추론 전력(power draw)은 입력 조건과 무관하게 모델 고유의 상수임을 밝히고, 에너지 변동의 실질적 원인이 출력 토큰 수에 있음을 분석했습니다. prefill(입력 처리)과 decode(출력 생성) 단계 간 compute-bound vs. memory-bound 비대칭성을 정량적으로 분석하여 에너지 병목 구조를 규명했습니다.

💡 *컨트리뷰션*: 기존 연구가 시각 토큰 감소에 집중했던 것과 달리, 출력 토큰 수가 에너지의 지배적 요인임을 실증적으로 밝혀 "보는 것은 공짜, 말하는 것이 비싸다"는 역발상의 통찰을 제시했습니다. 출력 길이를 제어하면 총 에너지의 최대 97%를 절감할 수 있음을 보였습니다.

🧪 *실험*: 1B~8B 파라미터 범위의 5개 모델을 대상으로 실험했으며, 모든 조건에서 추론 전력 변동은 5% 미만으로 나타났습니다. 각 출력 토큰은 입력 토큰 대비 11~39배 더 많은 처리 시간을 소모하며, 이미지 복잡도(객체 수)에 따라 동일 해상도에서도 최대 4.1배의 에너지 차이가 발생했습니다. 시각 토큰을 전부 제거하더라도 고정 토큰 모델 기준 총 에너지의 최대 10%만 절감되는 것으로 확인되었습니다.

⭐ **엣지 VLM 추론의 진짜 에너지 병목은 시각 처리가 아니라 출력 토큰 생성에 있으며, 출력 길이 제어가 에너지 절감의 핵심임을 최초로 체계적으로 증명한 연구**



📄 *2. VEXAIoT: Autonomous IoT Vulnerability EXploitation using AI Agents*

🔗 https://arxiv.org/abs/2607.09653

🏢 *기관/저자*: Katherine Swinea, Kshitiz Aryal, Lopamudra Praharaj, Maanak Gupta (소속 기관 미명시)

🔬 *메소드*: LLM 기반 추론과 오펜시브 보안 도구를 결합한 자율 멀티 에이전트 프레임워크(VEXAIoT)를 제안합니다. 취약점 탐지 에이전트(vulnerability detection agent)와 공격 실행 에이전트(attack execution agent)를 분리하여, 전자가 정찰(reconnaissance) 및 공격 시퀀스 계획을 담당하고 후자가 실제 익스플로잇을 수행합니다. OWASP IoT 취약점 분류 체계에 매핑된 10개의 공격 시나리오를 기반으로 파이프라인을 설계했습니다.

💡 *컨트리뷰션*: LLM 에이전트 기반 침투 테스트 연구가 CTF나 일반 환경에 집중했던 것과 달리, IoT 특화 취약점에 대한 자율 멀티 에이전트 프레임워크를 최초로 제안하고 실제 IoT 환경에서 검증했습니다.

🧪 *실험*: IoTGoat 및 Metasploitable2 환경에서 총 260회 공격 실행을 통해 평가했습니다. 전체 공격 성공률 95.0%(IoTGoat 94.5%, Metasploitable2 96.7%)를 달성했으며, 대부분의 공격이 평균 2분 이내에 완료되었습니다. 일부 시나리오에서는 100% 성공률을 기록했습니다.

⭐ **LLM 멀티 에이전트를 활용해 IoT 환경 취약점을 95% 성공률로 자율 탐지·익스플로잇하는 프레임워크를 제안한 연구**



📄 *3. What VGGT Knows About Overlap: Probing Geometric Foundation Models for Co-Visibility*

🔗 https://arxiv.org/abs/2607.09503

🏢 *기관/저자*: Filippo Ziliotto, Luciano Serafini, Lamberto Ballan, Tommaso Campari (소속 기관 미명시)

🔬 *메소드*: 기하학적 파운데이션 모델인 VGGT의 내부 레이어 표현을 분석하여, co-visibility(두 이미지가 공통 표면을 공유하는지 여부) 판단 능력이 감독 없이 창발적으로 학습됨을 규명했습니다. 이를 기반으로 VGGT를 동결(freeze)하고 레이어별 경량 Mixture-of-Experts(MoE) 헤드(7.5M 파라미터 미만)만을 학습시키는 Co-VGGT를 제안하며, 각 레이어를 독립적인 기하학적 전문가로 취급하여 입력 쌍에 따라 적응적으로 가중치를 부여합니다. RGB 이미지만을 입력으로 사용합니다.

💡 *컨트리뷰션*: VGGT 내부에서 초기 레이어는 3D 장면 표현을, 후기 레이어(특히 L17)는 co-visibility 판단을 전담하는 계층적 특화 구조가 존재함을 최초로 실증했으며, 이를 SfM/SLAM 파이프라인에 바로 활용 가능한 잘 보정된(calibrated) 예측기로 발전시켰습니다.

🧪 *실험*: Co-VisiON 벤치마크에서 인간 어노테이션 기준선을 초과하고, 선행 연구 대비 쌍별(pairwise) 정확도 25% 이상, 다중 뷰(multiview) 정확도 10% 이상 향상을 달성했습니다. 예측 보정 오차(ECE)는 0.030으로, 후처리 없이 가시성 그래프의 엣지 가중치로 직접 활용 가능한 수준입니다.

⭐ **VGGT의 내부 레이어에서 co-visibility 판단 능력이 자발적으로 창발됨을 발견하고, 경량 MoE 헤드만으로 SOTA를 크게 초월한 연구**



📄 *4. Communication-Efficient Digital-Twin Coordination for Heterogeneous LLM Embodied Agents over Computing Power Networks*

🔗 https://arxiv.org/abs/2607.09330

🏢 *기관/저자*: Nuocheng Yang, Sihua Wang, Zihan Chen, Tony Q. S. Quek, Changchuan Yin (소속 기관 미명시)

🔬 *메소드*: 이기종(heterogeneous) LLM 기반 구현 에이전트 팀의 협업을 위해 경량 디지털 트윈(Digital Twin, DT) 서버를 중재자로 활용하는 LDT-Coord 프레임워크를 제안합니다. 각 에이전트는 자연어 대화 없이 자신의 행동 결정과 공유 자원에 대한 구조화된 시간적 제약 조건만을 DT 서버에 보고하고, DT는 학습 불필요(training-free) 규칙 기반 오케스트레이터를 통해 충돌을 해소합니다. 통신 오버헤드 추가 절감을 위해 에이전트 보고 제어 문제를 제약 부분 관측 마르코프 결정 과정(C-POMDP)으로 모델링하고 PPO-Lagrangian 알고리즘으로 해결합니다.

💡 *컨트리뷰션*: 기존 다중 라운드 자연어 대화 기반 협업 방식의 통신 비용, LLM 성능 이질성, 행동 지연 문제를 동시에 해결하며, 협조 품질을 LLM의 언어 추론 능력으로부터 분리(decoupling)한 최초의 프레임워크입니다.

🧪 *실험*: 스마트 팩토리/창고/서비스 로봇 시나리오를 상정한 시뮬레이션 환경에서 평가했으며, 기존 협조 방식과 비교하여 동등한 작업 성공률을 유지하면서 통신 오버헤드를 70배 이상 절감했습니다. LLM 이질성 하에서도 안정적인 성능을 유지함을 확인했습니다.

⭐ **디지털 트윈 기반 규칙 오케스트레이터와 C-POMDP 보고 제어를 결합해 이기종 LLM 에이전트 협업의 통신 비용을 70배 이상 절감한 연구**



📄 *5. Revisiting Euler-Angle Regression with Kolmogorov-Arnold Networks*

🔗 https://arxiv.org/abs/2607.09650

🏢 *기관/저자*: Yangting Sun, Zijun Cui, Yufei Zhang (소속 기관 미명시)

🔬 *메소드*: 관절 로봇 및 생체역학 모델에서 자연스럽게 사용되는 오일러 각도(Euler angle) 회전 표현의 불연속성과 특이점(singularity) 문제를 해결하기 위해, 범위 인식(range-aware) 오일러 모델링과 Kolmogorov-Arnold Networks(KAN)를 결합한 새로운 프레임워크를 제안합니다. KAN은 고정된 노드별 활성화 함수 대신 엣지에 학습 가능한 단변수 함수를 사용하며, 이론적으로 유계(bounded) 오일러 범위가 회귀 함수의 근-덧셈(near-additive) 구조를 유도함을 분석했습니다. 이 근-덧셈 구조는 KAN의 덧셈적 함수 형태와 자연스럽게 부합합니다.

💡 *컨트리뷰션*: 오일러 각도 회귀에서 표현 방식, 회귀 아키텍처, 도메인 제약 간의 상호작용이 성능을 결정한다는 새로운 관점을 제시하고, KAN이 유계 오일러 범위 문제에 이론적·실증적으로 적합함을 최초로 규명했습니다.

🧪 *실험*: 제어된 회전 회귀, 객체 자세 추정(pose estimation), 로봇 역기구학(IK), 인간 역기구학 등 다양한 벤치마크에서 실험했으며, 기존 방법 대비 정확도, 수렴 속도, 효율성에서 일관된 성능 향상을 확인했습니다. (구체적 수치는 논문 본문 참조)

⭐ **KAN의 학습 가능한 단변수 함수가 유계 오일러 각도의 덧셈적 구조와 이론적으로 부합함을 밝히고, 로봇 및 자세 추정 과제에서 성능 향상을 실증한 연구**
