## 📝 상세 요약

📄 *1. GIFT: Geometry-Informed Low-precision Gradient Communication for LLM Pretraining*

🔗 https://arxiv.org/abs/2607.07494

🏢 *기관/저자*: Jieying Wang, Shuyuan Fan, Mingkai Zheng, Zhao Zhang (소속 기관 미공개)

🔬 *메소드*: GIFT는 LLM 사전학습 시 그래디언트를 FP8/NVFP4 등 저정밀도 포맷으로 통신하기 전에, 기하학적으로 인식된 좌표계로 변환하여 등방성(near-isotropic) 공간에서 양자화를 수행하는 방법론이다. 기존 유클리드 공간 기반 양자화는 고도로 비등방성인(anisotropic) 그래디언트에서 방향 왜곡이 심하게 발생하는 반면, GIFT는 저랭크 근사(low-rank approximation)와 선택적 적용을 통해 계산 오버헤드를 최소화하면서 좌표계 변환만을 수행한다. 옵티마이저, 학습 레시피, 통신 집합 연산, 저정밀도 포맷 자체는 변경하지 않아 기존 파이프라인과 완전히 호환된다.

💡 *컨트리뷰션*: 그래디언트의 기하학적 구조(비등방성)를 명시적으로 고려하여 저정밀도 통신의 충실도를 높인 최초의 기하학 인식 스케일링 방법으로, 기존 유클리드 FP8 통신 대비 다운스트림 태스크 성능 보존 프로파일을 개선하면서 통신 병목을 해소한다.

🧪 *실험*: Llama-300M 및 Llama-600M 모델에 대해 64개의 NVIDIA GH200 Superchip 환경에서 실험하였으며, Llama-600M 기준 엔드-투-엔드 사전학습 시간을 7.6% 단축하고, 동일한 옵티마이저 및 통신 경로 하에서 직접 유클리드 FP8 통신 대비 다운스트림 태스크 성능 보존 프로파일이 향상되었다.

⭐ **그래디언트의 비등방성을 기하학적으로 보정하는 좌표계 변환으로 LLM 사전학습의 저정밀도 통신 품질과 효율을 동시에 개선한 방법론**



📄 *2. Scaling Mixture-of-Experts Video Pretraining for Embodied Intelligence*

🔗 https://arxiv.org/abs/2607.07675

🏢 *기관/저자*: Shuailei Ma 외 다수 (소속 기관 미공개)

🔬 *메소드*: LingBot-Video는 DiT(Diffusion Transformer) 기반의 비디오 사전학습 패러다임으로, 밀집(dense) 아키텍처 대신 Mixture-of-Experts(MoE) 프레임워크를 채택하여 모델 용량과 추론 효율의 균형을 맞추고 스크래치부터 대규모 확장을 달성한다. 데이터 측면에서는 조작(manipulation), 탐색(navigation), 1인칭 시점(egocentric) 영상 등 로봇 지향 영상을 인터넷 비디오에 증강하는 데이터 프로파일링 엔진을 구축하였다. 학습 측면에서는 물리적 합리성과 태스크 완료 정렬을 강제하는 다차원 보상 시스템을 개발하여 미적 품질이나 프롬프트 추종 등 기존 기준을 넘어선다.

💡 *컨트리뷰션*: 콘텐츠 생성 중심의 기존 비디오 생성 모델이 가진 도메인 미스매치 문제를 해소하기 위해, MoE 구조와 로봇 특화 데이터·보상 시스템을 결합한 최초의 대규모 오픈소스 MoE 비디오 파운데이션 모델을 제시하여 디지털 창의성과 물리적 구동(physical actuation)을 연결한다.

🧪 *실험*: 비디오 파운데이션 모델로서의 성능과 효율에 대한 종합적 평가를 수행하였으며, 구체적인 수치는 논문 내 포괄적 평가(comprehensive evaluations)를 통해 모델 성능 및 추론 효율이 검증되었음을 보고하고 있다 (세부 벤치마크 수치는 초록에 명시되지 않음).

⭐ **MoE 아키텍처와 로봇 특화 데이터·보상 시스템을 결합하여 구현 효율성과 물리적 현실성을 갖춘 최초의 대규모 오픈소스 구현 지능용 비디오 파운데이션 모델**



📄 *3. Dual Latent Memory in Vision-Language-Action Models for Robotic Manipulation*

🔗 https://arxiv.org/abs/2607.07608

🏢 *기관/저자*: Hongyu Qu, Jianzhe Gao, Xiaobin Hu, Shaohuan Yang, Xinlei Yu, Rui Yan, Wenguan Wang, Xiangbo Shu, Shuicheng Yan (소속 기관 미공개)

🔬 *메소드*: LaMem-VLA는 과거 경험을 잠재 메모리 토큰(latent memory tokens)으로 재구성하여 VLA 추론 흐름에 직접 삽입하는 잠재 메모리 네이티브(latent-memory-native) 프레임워크이다. 핵심 구성요소로 ① 단기·장기 메모리 볼트를 조직하는 큐레이터(curator), ② 멀티모달 인지로 관련 증거를 조회하는 시커(seeker), ③ 검색된 증거를 컴팩트한 잠재 메모리 토큰으로 재구성하는 컨덴서(condenser), ④ 메모리 토큰을 현재 관찰·지시와 하나의 연속 임베딩 시퀀스로 통합하는 위버(weaver)의 네 가지를 제안한다. 모든 메모리 표현·검색·소비가 동일한 연속 잠재 공간에서 이루어져 메모리가 VLA 추론에 직접 참여한다.

💡 *컨트리뷰션*: 기존 메모리 증강 VLA가 메모리를 네이티브 잠재 공간 외부에 두는 한계를 극복하고, 단기·장기 이중 잠재 메모리를 멀티모달 추론 공간에 완전히 통합함으로써 장기 시계열 의존 태스크에서의 추론 및 행동 생성 품질을 향상시킨 점이 핵심 기여이다.

🧪 *실험*: SimplerEnv와 LIBERO 벤치마크에서 광범위한 실험을 수행하였으며, 두 벤치마크 모두에서 기존 방법론 대비 우수한 성능을 입증하였다 (구체적 수치는 초록에 명시되지 않음).

⭐ **과거 경험을 단기·장기 이중 잠재 메모리 토큰으로 재구성해 VLA 추론 흐름에 네이티브하게 통합한 장기 조작 태스크 특화 프레임워크**



📄 *4. EmbodiedGen V2: An Agentic, Simulation-Ready 3D World Engine for Embodied AI*

🔗 https://arxiv.org/abs/2607.07459

🏢 *기관/저자*: Xinjie Wang, Liu Liu, Taojun Ding, Andrew Choi, Chaodong Huang, Mengao Zhao, Ziang Li, Jackson Jiang, Chunlei Yu, Shengxiang Liu, Wei Xu, Zhizhong Su (소속 기관 미공개)

🔬 *메소드*: EmbodiedGen V2는 크로스 시뮬레이터 에셋, 상호작용 어포던스(affordance), 태스크 기반 월드, 대규모 멀티룸 씬, 상태 기반 Vibe Coding을 하나의 생성적·편집 가능·재사용 가능한 시뮬레이션 파이프라인으로 연결하는 통합 심-레디(sim-ready) 표현 방식을 채택한다. 기존에 수동으로 이루어지던 에셋 조립 과정을 에이전틱(agentic) 방식으로 자동화하여 조작, 탐색, 모바일 조작, 크로스 시뮬레이터 배포, 구현 정책 훈련을 모두 지원한다. 생성된 환경에서의 온라인 강화학습을 통해 정책을 학습하고 실제 로봇에 전이하는 전 과정을 하나의 파이프라인에서 처리한다.

💡 *컨트리뷰션*: 3D 에셋 생성에서 정책 학습 가능한 태스크 환경 조립까지의 갭을 자동화된 생성 엔진으로 메우고, 생성된 환경에서 학습한 정책이 실제 로봇에 효과적으로 전이됨을 정량적으로 입증한 대규모 오픈 시뮬레이션 인프라를 제공한다.

🧪 *실험*: 에셋 파이프라인에서 96.5% 인간 수용률(human acceptance)과 98.6% 충돌 성공률을 달성하였고, 태스크 기반 월드의 83.3%가 수동 수정 없이 바로 시뮬레이션에 사용 가능하였다. 온라인 강화학습 적용 시 시뮬레이션 성공률이 9.7%에서 79.8%로 향상되었으며, 실제 로봇 전이 후 태스크 성공률이 21.7%에서 75.0%로 대폭 증가하였다.

⭐ **3D 에셋 생성부터 정책 훈련·실제 로봇 전이까지 자동화하는 생성형 시뮬레이션 엔진으로, 시뮬레이션 성공률 9.7%→79.8%, 실제 로봇 성공률 21.7%→75.0% 달성**



📄 *5. TouchWorld: A Predictive and Reactive Tactile Foundation Model for Dexterous Manipulation*

🔗 https://arxiv.org/abs/2607.07287

🏢 *기관/저자*: Jianyi Zhou, Feiyang Hong, Yunhao Li, Yicheng Zhao, Yongjue Cen, Zirui Liu, Jiakang Huang, Zirui Chen, Ruiyang Zhang, Weizhuo Zhu, Xuhua Song, Shuo Yang (소속 기관 미공개)

🔬 *메소드*: TouchWorld는 예측(predictive)과 반응(reactive) 촉각 처리를 분리하는 계층적 정책 구조를 제안한다. 고수준 계획 레이어(High-Level Planning Layer)가 실행 가능한 서브태스크를 생성하고 촉각 서브골(tactile subgoal)을 예측하며, 비주오-촉각 골 조건부 정책(Visuo-Tactile Goal-Conditioned Policy)이 명목 행동 청크를 생성하고, 촉각 조건부 정제 정책(Tactile-Conditioned Refinement Policy)이 최근 촉각·고유감각 피드백으로 고주파 잔차 보정(online residual correction)을 수행한다. 이를 통해 느린 태스크 추론과 빠른 접촉 피드백을 단일 루프에 결합하는 기존 방식의 한계를 극복한다.

💡 *컨트리뷰션*: 촉각 센싱을 단순 저주파 관찰 스트림이 아닌 예측적 접촉 참조(predictive contact reference)이자 고주파 피드백 신호로 이중 활용하여, 비전-언어-행동 정책의 의미론적 일반화를 유지하면서 국소 접촉 적응력을 대폭 향상시킨 촉각 파운데이션 모델을 최초로 제시한다.

🧪 *실험*: 6개의 장기 시계열 접촉 집약적 정교 조작 태스크에서 실험하였으며,
