## 📝 상세 요약

📄 *1. RoboWorld: Fast and Reliable Neural Simulators for Generalist Robot Policy Evaluation*

🔗 https://arxiv.org/abs/2607.01060

🏢 *기관/저자*: Byeongguk Jeon, **Seonghyeon Ye** (KAIST), JaeHyeok Doo, Sungdong Kim, Minjoon Seo, Hyungmok Son, Kimin Lee

> ⚠️ **주목**: **Seonghyeon Ye** 참여 논문입니다.

🔬 *메소드*: RoboWorld는 빠른 자기회귀(autoregressive) 비디오 월드 모델과 태스크 진행 인식 비전-언어 모델(VLM) 스코어링을 결합한 자동화된 로봇 정책 평가 파이프라인이다. 핵심 기법인 **Step Forcing**은 앵커된 컨텍스트(anchored context)와 원스텝 자기전달(one-step self-forwarded) 컨텍스트를 혼합하여 학습-추론 간 분포 불일치(train-test mismatch)를 줄이면서도 액션-관측 다이나믹스를 보존한다. 이를 통해 장기 수평(long-horizon) 자기회귀 롤아웃의 신뢰성을 크게 향상시켰다.

💡 *컨트리뷰션*: 기존 비디오 월드 모델 기반 평가는 누적 오류와 느린 추론 속도로 인해 신뢰도가 낮았으나, RoboWorld는 Step Forcing으로 이를 해결하여 실제 로봇 평가와 거의 완벽한 상관관계를 달성한 최초의 신뢰성 높은 신경 시뮬레이터 평가 파이프라인이다.

🧪 *실험*: 다양한 태스크 및 환경에서 실제 로봇 평가 결과와의 상관관계를 측정한 결과, **Pearson's r = 0.989**, **Spearman's ρ = 0.970**을 달성하여 실세계 평가를 거의 완벽하게 대체할 수 있음을 입증하였다.

⭐ **비디오 월드 모델 기반 로봇 정책 평가의 신뢰성과 속도를 동시에 해결한 자동화 평가 파이프라인으로, 실제 로봇 평가와 r=0.989의 극도로 높은 상관관계를 달성했다.**



📄 *2. AutoMem: Automated Learning of Memory as a Cognitive Skill*

🔗 https://arxiv.org/abs/2607.01224

🏢 *기관/저자*: Shengguang Wu, Hao Zhu, Yuhui Zhang, Xiaohan Wang, Serena Yeung-Levy (Stanford University)

🔬 *메소드*: AutoMem은 LLM 에이전트의 메모리 관리를 학습 가능한 인지 기술로 취급하며, 파일 시스템 연산을 태스크 액션과 동등한 일급(first-class) 메모리 액션으로 승격시킨다. 두 개의 최적화 루프로 구성되어 있는데, **첫 번째 루프**는 강력한 LLM이 전체 에이전트 궤적을 검토하며 메모리 구조(프롬프트, 파일 스키마, 액션 어휘)를 반복적으로 개선하고, **두 번째 루프**는 에이전트 자신의 좋은 메모리 결정을 다수의 에피소드에서 식별하여 모델의 메모리 숙련도를 직접 향상시키는 학습 신호로 활용한다.

💡 *컨트리뷰션*: 메모리 관리를 독립적으로 학습 가능한 기술로 정의하고, 수천 스텝에 걸친 장기 수평 태스크에서 인간 검토 없이 두 축(구조 + 숙련도)을 자동으로 최적화하는 프레임워크를 최초로 제안하였다.

🧪 *실험*: Crafter, MiniHack, NetHack 세 가지 절차적 생성 장기 수평 게임에서 평가하였으며, 메모리 최적화만으로 기본 에이전트 성능을 **약 2x~4x 향상**시켰다. 32B 오픈 가중치 모델이 Claude Opus 4.5, Gemini 3.1 Pro Thinking 등 프론티어 시스템과 경쟁하는 수준에 도달하였다.

⭐ **메모리 관리를 독립적으로 학습 가능한 인지 기술로 다루는 AutoMem은, 32B 오픈 모델을 프론티어 LLM 수준으로 끌어올리는 2~4배의 성능 향상을 메모리 최적화만으로 달성했다.**



📄 *3. FurnitureVLA: Learning Long-Horizon Bimanual Furniture Assembly with Vision-Language-Action Model*

🔗 https://arxiv.org/abs/2607.01212

🏢 *기관/저자*: Chenyang Ma, Yue Yang, Radu Corcodel, Siddarth Jain, Andrew Wu, Chiori Hori, Diego Romeres (Mitsubishi Electric Research Laboratories, MERL)

🔬 *메소드*: FurnitureVLA는 실제 크기 양팔(bimanual) 가구 조립 태스크를 위해 VLA 모델을 의미론적으로 세분화된 서브태스크에 파인튜닝하고, 액션과 함께 **연속적인 진행도(progress) 신호**를 공동 예측하는 진행도 강화 VLA(progress-enhanced VLA)를 제안한다. VR 텔레오퍼레이션 시스템으로 고품질 실세계 시연 데이터를 수집하고, 시뮬레이션 파이프라인으로 전문가 데이터를 생성하여 확장 가능한 학습 환경을 구축하였다. 진행도 신호를 통해 자동 서브태스크 전환이 가능해져 최대 7개 서브태스크, 1,550 제어 스텝에 달하는 극단적 장기 수평 조립의 누적 오류를 완화한다.

💡 *컨트리뷰션*: VLA 모델을 활용한 실제 크기 양팔 가구 조립의 최초 체계적 연구로, 연속 진행도 신호를 통한 자동 서브태스크 전환 메커니즘을 도입하여 장기 수평 정밀 조작의 새로운 기준을 제시하였다.

🧪 *실험*: 세 가지 가구 유형에 대해 시뮬레이션 환경에서 기준 모델 대비 평균 성공률을 **48% → 80%**로 향상시켰으며, 설계 요인 연구를 통해 추가 21% 향상을 달성하였다. 실제 Kinova Gen3 플랫폼에서 가장 어려운 태스크 기준 시뮬레이션 대비 16% 성능 하락만으로 실세계 검증에 성공하였다.

⭐ **실제 크기 양팔 가구 조립에 VLA를 최초로 체계적으로 적용하여, 연속 진행도 신호 기반 서브태스크 전환으로 시뮬레이션 성공률 48%→80%를 달성하고 실세계 Kinova 로봇에서 검증했다.**



📄 *4. Task-Relevant Representation Decoupling for Visual Reinforcement Learning Generalization*

🔗 https://arxiv.org/abs/2607.00796

🏢 *기관/저자*: Jinwen Wang, Youfang Lin, Xiaobo Hu, Qian Xu, Shuo Wang, Zhuo Chen, Kai Lv (Beijing Jiaotong University 외)

🔬 *메소드*: T2RD(Task-Relevant Representation Decoupling)는 관측값을 태스크 관련(task-relevant) 표현과 태스크 무관(task-irrelevant) 표현으로 분리하는 자기지도(self-supervised) 알고리즘이다. **태스크 관련 표현 일관성(consistency)**과 **교차 재구성(cross-reconstruction)** 컴포넌트로 콘텐츠-스타일 피처를 분리하고, **교차 동적 예측(cross-dynamic prediction)** 컴포넌트를 추가하여 콘텐츠 표현에서 진정한 태스크 관련 피처만을 정제한다. 세 컴포넌트의 유기적 결합을 통해 에이전트가 환경 변화에 강건한 정책을 학습하도록 유도한다.

💡 *컨트리뷰션*: 기존 VRL 일반화 연구들이 스타일-콘텐츠 분리에 그쳤던 것과 달리, 동적 예측을 통해 콘텐츠 표현 중에서도 태스크 관련성을 추가로 정제하는 2단계 분리 개념을 도입하여 일반화와 샘플 효율성을 동시에 개선하였다.

🧪 *실험*: DeepMind Control Suite와 로봇 조작(Robotic Manipulation) 태스크에서 평가하였으며, 기존 SOTA 방법들 대비 일반화 성능 및 샘플 효율성 모두에서 최고 성능(SOTA)을 달성하였다.

⭐ **태스크 관련 표현을 자기지도 방식으로 분리하는 T2RD는 DeepMind Control Suite와 로봇 조작 태스크에서 VRL 일반화 SOTA를 달성하며, 시각적 강화학습의 환경 변화 대응력을 크게 높였다.**



📄 *5. ROSA: A Robotics Foundation Model Serving System for Robot Factories*

🔗 https://arxiv.org/abs/2607.01088

🏢 *기관/저자*: Wenqi Jiang, Jason Clemons, Rowland O'Flaherty, Hugo Hadfield, Alperen Degirmenci, Shuran Song, **Yashraj Narang** (NVIDIA GEAR), Christos Kozyrakis (Stanford University / NVIDIA)

> ⚠️ **주목**: **NVIDIA GEAR** 관련 논문입니다.

🔬 *메소드*: ROSA는 로봇 공장 환경을 위한 로보틱스 파운데이션 모델(RFM) 서빙 시스템으로, 세 가지 핵심 원칙을 기반으로 설계되었다. **공유 GPU 풀 서빙(shared GPU-pool serving)**으로 로봇 군집이 네트워크를 통해 강력한 서버급 GPU에 접근하게 하고, **로보틱스 인식 프로그래밍 추상화**로 멀티모델 파이프라인과 태스크별 성능 요구사항 및 장애 처리를 지원하며, **공장 목표 기반 스케줄링(factory-objective-driven scheduling)**으로 개별 요청 지연이 아닌 SLO(서비스 수준 목표) 기반 공장 생산성 극대화를 추구한다. Ray Serve 기반 분산 오케스트레이션 위에 vLLM, PyTorch, JAX를 백엔드로 구현되었다.

💡 *컨트리뷰션*: 기존 단일 로봇-단일 모델-엣지 컴퓨팅 패러다임에서 벗어나, 로봇 군집과 공유 GPU 풀을 연결하는 공장 규모의 RFM 서빙 시스템을 최초로 제안하고 구현하였다.

🧪 *실험*: 실제 로봇과 대규모 합성 워크로드 시뮬레이션 모두에서 평가하였으며, 기존 전용(dedicated) 서빙 시스템 대비 공장 생산성을 **최대 12.06배** 향상시키는 결과를 달성하였다.

⭐ **NVIDIA GEAR와 Stanford가 제안한 ROSA는 로봇 군집이 공유 GPU 풀을 활용하는 공장 규모 RFM 서빙 시스템으로, 기존 전용 서빙 대비 최대 12.06배의 공장 생산성 향상을 달성했다.**
