## 📝 상세 요약

📄 *1. WattGPU: Predicting Inference Power and Latency on Unseen GPUs and LLMs*

🔗 https://arxiv.org/abs/2607.02391

🏢 *Mauricio Fadel Argerich, Jonathan Fürst, Marta Patiño-Martínez* (소속 기관 미명시)

🔬 *메소드*: WattGPU는 LLM 추론 시 GPU 평균 전력 소비량과 토큰 간 지연(ITL)을 예측하는 두 가지 모델을 제안한다. 별도의 하드웨어 프로파일링 없이 공개된 LLM 메타데이터(파라미터 수 등)와 GPU 사양만을 입력으로 사용하며, leave-one-GPU-out 및 leave-one-LLM-out 교차 검증을 통해 학습 시 본 적 없는 GPU와 LLM에도 일반화할 수 있도록 설계되었다.

💡 *컨트리뷰션*: 기존 예측 모델들이 반드시 프로파일링 데이터를 요구하거나 미지의 하드웨어에 일반화하지 못했던 한계를 극복하고, 공개 메타데이터만으로 새로운 GPU-LLM 조합에 대한 전력 및 지연 예측이 가능함을 최초로 입증하였다.

🧪 *실험*: 42개의 오픈소스 LLM(0.1B~27B 파라미터)과 8개의 NVIDIA 서버급 GPU 조합을 대상으로 오프라인/서버 두 시나리오에서 평가. 전력 예측 모델은 미지의 GPU에 대해 중앙값 절대 백분율 오차(MdAPE) ≤3.4%(오프라인), ≤13.5%(서버)를 달성했으며, 지연 예측 모델은 서버 모드에서 ≤8.5%를 기록. 서버 시나리오에서 Kendall τ ≥ 0.76의 GPU 랭킹 상관관계를 유지하며, TDP 기반·루프라인 기반 물리적 베이스라인 대비 MdAPE를 약 4배(서버) 또는 약 2배(미지 GPU) 감소시켰다.

⭐ **하드웨어 프로파일링 없이 공개 메타데이터만으로 미지의 GPU·LLM 조합의 전력 및 지연을 높은 정확도로 예측하는 경량 예측 프레임워크**



📄 *2. Automated grading of Linux/bash examinations using large language models: a four-level cognitive taxonomy approach*

🔗 https://arxiv.org/abs/2607.02432

🏢 *Manuel Alonso-Carracedo, Ruben Fernandez-Boullon, Pedro Celard, Francisco J. Rodriguez-Martinez, Lorena Otero-Cerdeira* (소속 기관 미명시)

🔬 *메소드*: GPT, Claude Opus, Gemini, GLM 4개의 프론티어 LLM을 활용하여 Linux/bash 명령어 시험 답안을 자동 채점하는 방법론을 제안한다. 인지적 복잡도와 운영적 영향도를 결합한 4단계 인지 분류 체계(L1: 정보 검색, L2: 기본 파일 조작, L3: 구조적 연산, L4: 고급 시스템 관리)를 도입하였으며, 최소 프롬프트와 루브릭 강화 프롬프트 두 가지 변형으로 실험하였다.

💡 *컨트리뷰션*: 단순 정답/오답 판별을 넘어 부분 점수, 동등한 대체 명령어, 구문 변형을 처리할 수 있는 분류 기반 AI 채점 프레임워크를 제시하고, 질문 복잡도가 LLM 채점 난이도의 신뢰할 수 있는 예측 지표임을 처음으로 체계적으로 규명하였다.

🧪 *실험*: 컴퓨터공학과 2학년 학생 1,200개의 실제 답변을 3인의 전문가 채점과 비교. Gemini 3.0 Pro + 루브릭 프롬프트가 최고 인간-AI 일치도(ICC(3,1) = 0.888, MAE = 0.10, Bland-Altman 편향 = -0.014)를 달성. 분류 수준이 높을수록 일치도가 일관되게 하락하였으며, 모델 제공사보다 루브릭 품질이 성능에 더 큰 영향을 미쳤다.

⭐ **4단계 인지 분류 체계 기반 LLM 자동 채점 프레임워크로, 루브릭 설계가 모델 선택보다 더 중요한 성능 결정 인자임을 입증**



📄 *3. Embodied.cpp: A Portable Inference Runtime of Embodied AI Models on Heterogeneous Robots*

🔗 https://arxiv.org/abs/2607.02501

🏢 *Ling Xu, Chuyu Han, Borui Li, Hao Wu, Shiqi Jiang, Ting Cao, Chuanyou Li, Sheng Zhong, Shuai Wang* (소속 기관 미명시)

🔬 *메소드*: Embodied.cpp는 VLA(Vision-Language-Action) 모델과 WAM(World-Action Model)을 이기종 로봇 엣지 디바이스에 배포하기 위한 포터블 C++ 추론 런타임이다. 대표적인 VLA 및 WAM 아키텍처 분석을 바탕으로 공유 실행 경로를 입력 어댑터, 시퀀스 빌더, 백본 실행, 헤드 플러그인, 배포 어댑터의 5개 계층으로 추상화하였으며, 다중 속도 실행(multi-rate execution), 지연 우선 융합 추론, 확장 가능한 연산자 및 I/O 지원을 제공한다.

💡 *컨트리뷰션*: 기존 추론 런타임이 요청-응답 서빙에 특화되어 폐루프 제어의 실시간 요건을 충족하지 못했던 문제를 해결하고, 단일 백엔드 추상화로 다양한 이기종 디바이스·로봇·시뮬레이터에 걸쳐 VLA와 WAM을 통합 배포할 수 있는 최초의 범용 C++ 런타임을 제시하였다.

🧪 *실험*: HY-VLA와 pi0.5 두 VLA 모델에 대해 폐루프 실행 평가를 수행한 결과 각각 100.0% 및 91.0%의 태스크 성공률을 달성. WAM 벤치마크(LingBot-VA Transformer 블록)에서는 블록 메모리를 312.2 MiB에서 88.1 MiB로 약 71.8% 감소시켰다.

⭐ **이기종 로봇 엣지 디바이스를 위한 포터블 C++ 추론 런타임으로, 단일 백엔드로 VLA·WAM 모델의 실시간 폐루프 배포와 대폭적인 메모리 절감을 동시에 실현**



📄 *4. Learning to Move Before Learning to Do: Task-Agnostic pretraining for VLAs*

🔗 https://arxiv.org/abs/2607.02466

🏢 *Junhao Shi, Siyin Wang, Xiaopeng Yu, Li Ji, Jingjing Gong, Xipeng Qiu* (소속 기관 미명시)

🔬 *메소드*: TAP(Task-Agnostic Pretraining)은 VLA 학습을 "어떻게 움직이는가(물리적 역량)"와 "무엇을 해야 하는가(의미론적 정렬)"로 분리하는 분해 가설(Decomposition Hypothesis)에 기반한 2단계 프레임워크다. 1단계에서는 폐기된 오프태스크 궤적이나 자율적 로봇 플레이 데이터 등 라벨 없는 저비용 상호작용 데이터로부터 자기지도 역 동역학(Inverse Dynamics) 목적 함수를 통해 전이 가능한 운동 사전(motor prior)을 학습하고, 2단계에서는 소량의 전문가 데이터만으로 언어와 운동 사전을 정렬한다.

💡 *컨트리뷰션*: 언어 지도가 필요 없는 저비용 인터랙션 데이터로 물리적 역량을 먼저 학습함으로써 고비용 전문가 시연 데이터 의존도를 획기적으로 낮추는 확장 가능한 VLA 사전학습 패러다임을 제시하였다.

🧪 *실험*: SIMPLER 벤치마크에서 TAP은 표준 행동 복제(BC) 대비 절대 성능 10% 향상을 달성하며 1M개 이상의 전문가 궤적으로 학습된 모델과 동등한 성능을 훨씬 적은 라벨 데이터로 달성. 실제 WidowX 플랫폼에서 카메라 교란 조건 하에 TAP은 25% 성공률을 유지한 반면, 인터넷 규모 베이스라인은 0%로 완전히 실패하였다.

⭐ **"움직임을 먼저, 의미는 나중에" 학습하는 2단계 TAP 프레임워크로, 소량의 전문가 데이터로도 강인하고 전이 가능한 로봇 운동 표현을 획득**



📄 *5. WorldSample: Closed-loop Real-robot RL with World Modelling*

🔗 https://arxiv.org/abs/2607.02431

🏢 *Yuquan Xue, Le Xu, Zeyi Liu, Zhenyu Wu, Zhengyi Gu, Xinyang Song, Bofang Jia, Ziwei Wang* (소속 기관 미명시)

🔬 *메소드*: WorldSample은 실제 로봇 강화학습을 위한 물리적으로 그라운딩된 데이터 증강 프레임워크로, 실제 롤아웃 → 월드 모델 생성 → 정책 개선의 실-합성(real-synthetic) 폐루프를 구성한다. 실제 롤아웃 데이터를 기반으로 사후 학습된(post-trained) 월드 모델이 고충실도 합성 전이(transition)를 생성하여 시각적 환각을 최소화하며, Policy-Paced Learning(PPL)을 통해 샘플 선택 및 스케줄링으로 유용한 증강과 가치 과대추정 사이의 균형을 조절한다.

💡 *컨트리뷰션*: 단순히 합성 전이를 실제 경험으로 대체하는 기존 접근과 달리, PPL을 통해 월드 모델의 환각으로 인한 노이즈를 능동적으로 제어하는 실-합성 폐루프를 도입함으로써 정책과 월드 모델 성능을 동시에 향상시켰다.

🧪 *실험*: 접촉이 많고 정밀한 로봇 조작 태스크에서 평가. WorldSample은 베이스라인 대비 정책 성공률 28% 향상, 학습 스텝 59% 감소를 달성. 월드 모델 시각 충실도도 시연 전용 사후 학습 대비 PSNR 19.4dB 향상, SSIM 0.47 향상을 기록하였다.

⭐ **실-합성 폐루프와 Policy-Paced Learning으로 실제 로봇 RL의 샘플 효율과 월드 모델 충실도를 동시에 대폭 개선한 데이터 증강 프레임워크**
